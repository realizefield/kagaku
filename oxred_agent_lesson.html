酸化剤・還元剤

主な酸化剤8種・還元剤6種の反応前後と生成物を学びます。
色の変化・条件・なぜその生成物になるかの理由まで、しっかり理解しましょう。

⚗️ 第3章　色付き解説
0酸化剤・還元剤とは

前章で「酸化還元の本質は電子のやり取り」と学びました。
その「電子を渡す・受け取る」という役割を担う物質に名前があります。

酸化剤
相手から電子を受け取る
→ 相手を酸化させる
自分自身は還元される
（酸化数が減る）
還元剤
相手に電子を渡す
→ 相手を還元させる
自分自身は酸化される
（酸化数が増える）
⚠️ 名前と逆の変化に注意！
「酸化剤」という名前なのに、自分自身は還元される（酸化数が減る）。
「還元剤」という名前なのに、自分自身は酸化される（酸化数が増える）。
→「酸化剤は相手を酸化させる道具」であって、自分が酸化されるわけではありません。
★ セットで覚える
  • 酸化剤 → 電子を受け取る → 自分は還元される → 自分の酸化数が減る
  • 還元剤 → 電子を渡す → 自分は酸化される → 自分の酸化数が増える
✏️ 確認問題（全2問）
問1酸化剤が反応するとき、酸化剤自身はどうなるか。
正解はイ：還元される。
酸化剤は相手から電子を受け取る → 自分の酸化数が減る → 自分は還元される。
「酸化剤なのに還元される」という逆の関係が重要です。
問2還元剤が電子を渡すとき、還元剤自身の酸化数はどうなるか。
正解はウ：増える。
還元剤は電子を渡す → 電子が減る → 酸化数が増える → 自分は酸化される。

✏️ 酸化剤の確認（全4問）
問1KMnO₄ が酸性条件下で酸化剤としてはたらくとき、生成するイオンと色は何か。
正解はア：Mn²⁺（淡桃色〜ほぼ無色）。
KMnO₄（赤紫色） → 反応後 Mn²⁺（ほぼ無色）。
Mn の酸化数：+7 → +2（5減る）。この色変化が滴定の終点判定に使われます。
問2K₂Cr₂O₇ が酸化剤としてはたらいた後、Cr³⁺ の色は何色か。
正解はウ：緑色。
Cr₂O₇²⁻（橙赤色） → Cr³⁺（緑色）。
Cr の酸化数：+6 → +3（3減る × 2個 = e⁻ 6個）。色変化もセットで覚えましょう。
問3希硝酸と濃硝酸で生成する気体の違いとして正しいものはどれか。
正解はイ。
希 HNO₃ → NO（無色、N：+5→+2）
濃 HNO₃ → NO₂（赤褐色）（N：+5→+4）
この区別は入試頻出です。「希→NO、濃→NO₂」とセットで。
問4次のうち酸化剤として正しいものはどれか。
正解はエ：Cl₂。
Cl₂ は e⁻ を受け取り Cl⁻ になる酸化剤。
希硫酸は酸化剤にならない（酸として反応するのみ）。H₂S とシュウ酸は還元剤です。

✏️ 還元剤の確認（全4問）
問1H₂S が還元剤としてはたらくとき、生成する物質と色は何か。
正解はイ：S（黄色の固体）。
H₂S の S：−2 → 0（2増える → e⁻ 2個渡す → 還元剤）。
反応後は 硫黄（黄色の固体）が析出します。
問2SO₂ が還元剤としてはたらくとき、生成するのは何か。
正解はエ：SO₄²⁻。
SO₂（S：+4）→ SO₄²⁻（S：+6）。S の酸化数が増えた → 酸化された → 還元剤。
SO₂ が酸化剤のとき（H₂S との反応）は S（単体）になりますが、還元剤のときは SO₄²⁻ です。
問3シュウ酸 (COOH)₂ が還元剤としてはたらくとき、生成するのは何か。
正解はウ：CO₂（二酸化炭素）。
(COOH)₂ の C：+3 → CO₂ の C：+4（1増える × 2 = e⁻ 2個渡す）。
1mol のシュウ酸から e⁻ 2mol を渡します。KMnO₄ 滴定の計算に重要です。
問4H₂O₂ が還元剤としてはたらくのはどのときか。
正解はア。
H₂O₂ は通常酸化剤（→ H₂O または OH⁻）ですが、
KMnO₄ のような超強力な酸化剤と出会ったときだけ還元剤（→ O₂）になります。
KI との反応では H₂O₂ が酸化剤として I⁻ を I₂ に酸化します。

❓ KMnO₄ ─ なぜ Mn²⁺ ができるのか

Mn の酸化数の変化を追う

KMnO₄ の Mn の酸化数は +7（最大付近）です。
酸化剤として電子を受け取ると、酸化数が下がります。

MnO₄⁻（+7）+ 8H⁺ + 5e⁻ → Mn²⁺（+2）+ 4H₂O

①Mn：+7 → +2（5減る → e⁻ を5個受け取る）

②O が4個あるので右辺に H₂O を4個加える

③H を合わせるために左辺に H⁺ を8個加える

④電荷を合わせるために左辺に e⁻ を5個加える

酸性条件が必要な理由

H⁺ が8個必要なため、硫酸酸性にして H⁺ を供給します。
中性・塩基性では H⁺ が不足し、代わりに MnO₂（黒色の沈殿）が生成します。

酸性 → Mn²⁺（淡桃〜無色）｜中性・塩基性 → MnO₂（黒色）
色の変化が終点のサイン：
赤紫色（MnO₄⁻） → 反応中は脱色 → 終点で淡桃色が残る

❓ K₂Cr₂O₇ ─ なぜ Cr³⁺ になるのか

Cr の酸化数の変化

Cr₂O₇²⁻ の Cr は酸化数 +6。
酸化剤として電子を受け取ると +3 に下がります。

Cr₂O₇²⁻（+6）+ 14H⁺ + 6e⁻ → 2Cr³⁺（+3）+ 7H₂O

①Cr が 2個あるので：1個あたり 3 減る × 2個 = e⁻ 6個受け取る

②O が7個あるので右辺に H₂O を7個加える

③H を合わせるために左辺に H⁺ を14個加える

④電荷を合わせるために左辺に e⁻ を6個加える

Cr₂O₇²⁻（橙赤色） → Cr³⁺（緑色）
塩基性では CrO₄²⁻（黄色）が安定（酸性⇌塩基性で橙⇌黄と変化）

❓ H₂O₂ ─ なぜ酸性と塩基性で生成物が違うのか

カギは「H⁺ が使えるかどうか」

H₂O₂ が e⁻ を受け取ると O の酸化数が −1 → −2 になります。
この「−2 になった O」が H⁺ を拾えるかどうかで生成物が変わります。

① 酸性条件（H⁺ が豊富にある）

H₂O₂ + 2H⁺ + 2e⁻ → 2H₂O

O²⁻ が外から H⁺ を2個拾える → OH⁻ + H⁺ → H₂O まで進める

💡 H⁺ が豊富 → O が H⁺ を2個引き寄せて H₂O になる

② 中性・塩基性条件（H⁺ がほとんどない）

H₂O₂ + 2e⁻ → 2OH⁻

H⁺ が来ないので H₂O₂ 自身の H をそのまま使う
→ H−O⁻ の形、つまり OH⁻ のまま止まる

💡 H⁺ が不足 → OH⁻ のまま止まる
一言まとめ：
「OH⁻ に H⁺ を渡せるかどうかの違い」
酸性 → H⁺ がある → H₂O へ　｜　中性・塩基性 → OH⁻ のまま

❓ H₂O₂ ─ なぜ二面性を持つのか

カギは O の酸化数 −1 が「中間値」であること

酸素の酸化数は通常 −2（最小値）ですが、
H₂O₂ の O は −1（中間値） という例外的な値をとります。

物質O の酸化数動ける方向
H₂O・OH⁻−2（最小）これ以上下がれない
H₂O₂−1（中間）↓−2 にも ↑0 にもなれる
O₂0（最大）これ以上上がれない

通常（酸化剤）：−1 → −2 へ下がる

H₂O₂ + 2H⁺ + 2e⁻ → 2H₂O

相手より H₂O₂ の方が電子を引き寄せる力が強い → e⁻ を受け取る → 酸化剤

KMnO₄ と反応するとき（還元剤）：−1 → 0 へ上がる

H₂O₂ → O₂ + 2H⁺ + 2e⁻

KMnO₄（Mn：+7）の方が圧倒的に電子を引き寄せる力が強い
→ H₂O₂ が逆に e⁻ を渡す側に回る → 還元剤

一言まとめ：
O の酸化数 −1 は中間値 → 上にも下にも動ける
KMnO₄ との反応のときだけ → O が 0 の方向へ → 還元剤（→ O₂）
それ以外はすべて → O が −2 の方向へ → 酸化剤（→ H₂O または OH⁻）

❓ SO₂ ─ なぜ二面性を持つのか

カギは S の酸化数 +4 が「中間値」であること

硫黄 S の酸化数は −2 〜 +6 の範囲で変化できます。
SO₂ の S は +4 で、ちょうど中間付近にあります。

物質S の酸化数特徴
H₂S−2（最小）電子を渡すしかない → 強力な還元剤
SO₂+4（中間）上にも下にも動ける → 二面性あり
H₂SO₄（濃）+6（最大）電子を受け取るしかない → 酸化剤のみ

還元剤になるとき（通常）：+4 → +6 へ上がる

SO₂ + 2H₂O → SO₄²⁻ + 4H⁺ + 2e⁻

KMnO₄・I₂ などの強い酸化剤と出会う → SO₂ が e⁻ を渡す → 還元剤

酸化剤になるとき（H₂S との反応）：+4 → 0 へ下がる

2H₂S + SO₂ → 3S + 2H₂O

H₂S の S（−2）は最底辺 → 電子を渡す力が SO₂ より圧倒的に強い
→ SO₂ が「やむなく」e⁻ を受け取る側に → 酸化剤

力関係で決まる：
強い酸化剤（KMnO₄・I₂ など）と出会う → SO₂ は還元剤（S：+4→+6）
強い還元剤（H₂S）と出会う → SO₂ は酸化剤（S：+4→0）
