半反応式の組み合わせ

酸化剤と還元剤それぞれの半反応式を組み合わせて、イオン反応式を作る方法を学びます。
電子（e⁻）の数を揃えることがポイントです。

⚗️ 第4章　入試頻出6例
0イオン反応式を作る4ステップ

酸化剤と還元剤が反応するとき、酸化剤が受け取るe⁻の数と還元剤が渡すe⁻の数は必ず等しくなります。この関係を使って2つの半反応式を組み合わせ、イオン反応式を作ります。

📋 イオン反応式を作る4ステップ
①
酸化剤の半反応式を書く（e⁻ を左辺に）
例：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
②
還元剤の半反応式を書く（e⁻ を右辺に）
例：Fe²⁺ → Fe³⁺ + e⁻
③
e⁻ の係数を揃えるように両辺を整数倍にする
例：酸化剤のe⁻ = 5mol、還元剤のe⁻ = 1mol → 還元剤の式を5倍にする
④
2つの式を辺々加える（e⁻ が消える）
→ イオン反応式の完成！
⚠️ 注意
両辺に同じ物質が現れる場合は消去します（例：H⁺・H₂O・共通イオン）。
また、イオン反応式から化学反応式に直すときはイオンを元の物質に戻します。
1KMnO₄（酸化剤）＋ H₂O₂（還元剤）　酸性条件

過マンガン酸カリウムと過酸化水素の反応。KMnO₄ が酸化剤、H₂O₂ が還元剤としてはたらきます。H₂O₂ は酸化剤にも還元剤にもなる二面性の試薬なので、相手によって役割が変わります。

⚗️ KMnO₄ + H₂O₂ → イオン反応式を作ろう
酸化剤：KMnO₄（e⁻ 5個） 還元剤：H₂O₂（e⁻ 2個）
Step 0：半反応式を書く
酸化剤 MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
還元剤 H₂O₂ → O₂ + 2H⁺ + 2e⁻
酸化剤 MnO₄⁻ は e⁻ 5個 を受け取ります。
還元剤 H₂O₂ は e⁻ 2個 を渡します。
→ このままでは e⁻ の数が合わない → 係数を調整します。
Step 1：e⁻ の数を揃える
🔢 最小公倍数で揃える
酸化剤のe⁻：5個　還元剤のe⁻：2個
最小公倍数 = 10
酸化剤の式 × 2　（5e⁻ × 2 = 10e⁻）
還元剤の式 × 5　（2e⁻ × 5 = 10e⁻）
酸化剤×2 2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + 8H₂O
還元剤×5 5H₂O₂ → 5O₂ + 10H⁺ + 10e⁻
両式とも e⁻ が 10個 になりました。次に2つの式を足します。
Step 2：辺々加える（e⁻ を消去）
＋ 辺々加える
2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + 8H₂O
5H₂O₂ → 5O₂ + 10H⁺ + 10e⁻
2MnO₄⁻ + 16H⁺ + 5H₂O₂ + 10e⁻ → 2Mn²⁺ + 8H₂O + 5O₂ + 10H⁺ + 10e⁻
e⁻ が両辺に同数あるので消えます。次に H⁺ と H₂O も整理します。
Step 3：共通項を整理（H⁺ を消去）
両辺の H⁺ を整理（左辺 16H⁺ − 右辺 10H⁺ = 6H⁺ が左辺に残る）
2MnO₄⁻ + 6H⁺ + 5H₂O₂ → 2Mn²⁺ + 8H₂O + 5O₂
左辺の H⁺（16個）から右辺の H⁺（10個）を引いて 6H⁺ が左辺に残ります。
H₂O はそのまま（左右で消えるものはない）。
完成！イオン反応式
2MnO₄⁻ + 6H⁺ + 5H₂O₂ → 2Mn²⁺ + 8H₂O + 5O₂
左辺：酸化剤 + 酸（H⁺）+ 還元剤　→　右辺：生成物
  • MnO₄⁻（赤紫色）→ Mn²⁺（ほぼ無色）：色変化が滴定の終点の目印
  • H₂O₂（KMnO₄と反応するとき）→ O₂（H₂O₂ は還元剤として機能）
  • この反応はH₂O₂の定量分析（過酸化水素の濃度決定）に使われる
✏️ 確認問題
問1KMnO₄ + H₂O₂ の反応で、イオン反応式の H⁺ の係数はいくつか。
正解はウ：6。
酸化剤×2 → 16H⁺、還元剤×5 → 10H⁺（右辺）。16 − 10 = 6H⁺ が左辺に残ります。
問2この反応でH₂O₂がはたらく役割はどれか。
正解はイ：還元剤（→ O₂）。
H₂O₂ は通常酸化剤ですが、KMnO₄ のような強い酸化剤と反応するときは還元剤になります（H₂O₂ の二面性）。
2KMnO₄（酸化剤）＋ (COOH)₂（還元剤）　酸性条件

シュウ酸滴定（KMnO₄ の標定）に使われる重要な反応です。シュウ酸は基準試薬（一次標準物質）として KMnO₄ の正確な濃度を決定するのに用います。

⚗️ KMnO₄ + (COOH)₂ → イオン反応式を作ろう
酸化剤：KMnO₄（e⁻ 5個） 還元剤：(COOH)₂（e⁻ 2個）
Step 0：半反応式を書く
酸化剤 MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
還元剤 C₂O₄²⁻ → 2CO₂ + 2e⁻
シュウ酸（シュウ酸イオン C₂O₄²⁻）は e⁻ 2個を渡す還元剤。
KMnO₄ との比率：e⁻の最小公倍数 = 10 → 酸化剤×2、還元剤×5
Step 1：e⁻ の数を揃える（最小公倍数10）
酸化剤×2 2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + 8H₂O
還元剤×5 5C₂O₄²⁻ → 10CO₂ + 10e⁻
Step 2：辺々加えて e⁻ を消去
＋ 辺々加える（e⁻ 消去）
2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + 8H₂O
5C₂O₄²⁻ → 10CO₂ + 10e⁻
2MnO₄⁻ + 16H⁺ + 5C₂O₄²⁻ → 2Mn²⁺ + 8H₂O + 10CO₂
e⁻ が消え、H⁺ や H₂O の整理も不要。このままで完成です。
完成！イオン反応式
2MnO₄⁻ + 16H⁺ + 5C₂O₄²⁻ → 2Mn²⁺ + 8H₂O + 10CO₂
シュウ酸滴定の基本式　KMnO₄ : (COOH)₂ = 2 : 5（mol比）
  • KMnO₄ 2mol と (COOH)₂ 5mol が過不足なく反応（2:5 の mol 比）
  • この反応は加熱しながら行う（シュウ酸との反応は低温では遅い）
  • 終点：赤紫色（KMnO₄）が消えて薄い桃色が残る
✏️ 確認問題
問1KMnO₄ と (COOH)₂ の mol 比（KMnO₄：(COOH)₂）はいくらか。
正解はエ：2：5。
イオン反応式より 2MnO₄⁻：5C₂O₄²⁻ → KMnO₄：(COOH)₂ = 2：5。
e⁻ で確認：KMnO₄（e⁻ 5個）×2 = 10 ＝ (COOH)₂（e⁻ 2個）×5 = 10 ✓
3KMnO₄（酸化剤）＋ FeSO₄（還元剤）　酸性条件

FeSO₄（硫酸鉄(II)）は1mol で e⁻ を1mol 渡す還元剤です。e⁻ の最小公倍数が5になるため、係数がシンプルな入試頻出の反応です。

⚗️ KMnO₄ + FeSO₄ → イオン反応式を作ろう
酸化剤：KMnO₄（e⁻ 5個） 還元剤：FeSO₄（e⁻ 1個）
Step 0：半反応式を書く
酸化剤 MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
還元剤 Fe²⁺ → Fe³⁺ + e⁻
FeSO₄ の Fe²⁺ は e⁻ 1個を渡す。e⁻ の最小公倍数 = 5 → 還元剤を5倍。
Step 1：e⁻ の数を揃える（最小公倍数5）
酸化剤×1 MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
還元剤×5 5Fe²⁺ → 5Fe³⁺ + 5e⁻
Step 2：辺々加えて完成
＋ 辺々加える（e⁻ 消去）
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
5Fe²⁺ → 5Fe³⁺ + 5e⁻
MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺
完成！イオン反応式
MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺
KMnO₄ : FeSO₄ = 1 : 5（mol比）
  • KMnO₄ 1mol と FeSO₄ 5mol が反応（1：5 の mol 比）
  • 係数がシンプルで覚えやすい反応
✏️ 確認問題
問1KMnO₄ と FeSO₄ の mol 比（KMnO₄：FeSO₄）はいくらか。
正解はウ：1：5。
KMnO₄（e⁻ 5個）: FeSO₄（e⁻ 1個）→ 電子の数を揃えると 1×5 = 5×1 → 1：5。
4K₂Cr₂O₇（酸化剤）＋ FeSO₄（還元剤）　酸性条件

ニクロム酸カリウムと硫酸鉄(II)の反応。Cr₂O₇²⁻ は e⁻ 6個を受け取る（Cr が2個で各3個）。FeSO₄ は e⁻ 1個を渡すため、還元剤を6倍します。

⚗️ K₂Cr₂O₇ + FeSO₄ → イオン反応式を作ろう
酸化剤：K₂Cr₂O₇（e⁻ 6個） 還元剤：FeSO₄（e⁻ 1個）
Step 0：半反応式を書く
酸化剤 Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
還元剤 Fe²⁺ → Fe³⁺ + e⁻
Cr₂O₇²⁻ は e⁻ 6個を受け取る。e⁻ の最小公倍数 = 6 → 還元剤を6倍。
Step 1：e⁻ の数を揃える（最小公倍数6）
酸化剤×1 Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
還元剤×6 6Fe²⁺ → 6Fe³⁺ + 6e⁻
Step 2：辺々加えて完成
＋ 辺々加える（e⁻ 消去）
Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
6Fe²⁺ → 6Fe³⁺ + 6e⁻
Cr₂O₇²⁻ + 14H⁺ + 6Fe²⁺ → 2Cr³⁺ + 7H₂O + 6Fe³⁺
完成！イオン反応式
Cr₂O₇²⁻ + 14H⁺ + 6Fe²⁺ → 2Cr³⁺ + 7H₂O + 6Fe³⁺
K₂Cr₂O₇ : FeSO₄ = 1 : 6（mol比）
  • K₂Cr₂O₇ 1mol と FeSO₄ 6mol が反応（1：6 の mol 比）
  • 色変化：橙赤色（Cr₂O₇²⁻）→ 緑色（Cr³⁺）
✏️ 確認問題
問1K₂Cr₂O₇ と FeSO₄ の mol 比（K₂Cr₂O₇：FeSO₄）はいくらか。
正解はイ：1：6。
Cr₂O₇²⁻（e⁻ 6個）：Fe²⁺（e⁻ 1個）→ e⁻ を揃えると 1×6 = 6×1 → 1：6。
5Cl₂（酸化剤）＋ FeSO₄（還元剤）

塩素は e⁻ 2個を受け取る酸化剤、FeSO₄ は e⁻ 1個を渡す還元剤。最小公倍数が2のシンプルな反応です。H₂O も H⁺ も関係しない点が特徴です。

⚗️ Cl₂ + FeSO₄ → イオン反応式を作ろう
酸化剤：Cl₂（e⁻ 2個） 還元剤：FeSO₄（e⁻ 1個）
Step 0：半反応式を書く
酸化剤 Cl₂ + 2e⁻ → 2Cl⁻
還元剤 Fe²⁺ → Fe³⁺ + e⁻
Cl₂ は H₂O も H⁺ も不要なシンプルな式。e⁻ の最小公倍数 = 2 → 還元剤を2倍。
Step 1：e⁻ の数を揃える（最小公倍数2）
酸化剤×1 Cl₂ + 2e⁻ → 2Cl⁻
還元剤×2 2Fe²⁺ → 2Fe³⁺ + 2e⁻
Step 2：辺々加えて完成
＋ 辺々加える（e⁻ 消去）
Cl₂ + 2e⁻ → 2Cl⁻
2Fe²⁺ → 2Fe³⁺ + 2e⁻
Cl₂ + 2Fe²⁺ → 2Cl⁻ + 2Fe³⁺
完成！イオン反応式
Cl₂ + 2Fe²⁺ → 2Cl⁻ + 2Fe³⁺
Cl₂ : FeSO₄ = 1 : 2（mol比）　H₂O も H⁺ も不要
  • Cl₂ 1mol と FeSO₄ 2mol が反応（1：2 の mol 比）
  • H₂O も H⁺ も登場しない非常にシンプルな式
  • 色変化：黄緑色（Cl₂）→ 無色（Cl⁻）
✏️ 確認問題
問1Cl₂ + FeSO₄ のイオン反応式として正しいのはどれか。
正解はア。
Cl₂（e⁻ 2個）+ 2Fe²⁺（e⁻ 各1個 × 2）→ H₂O も H⁺ も不要。係数は Cl₂：Fe²⁺ = 1：2。
6HNO₃（希）（酸化剤）＋ Cu（還元剤）

希硝酸と銅の反応。銅は希塩酸・希硫酸には溶けないが、希硝酸には溶ける重要な反応です。銅の単体は e⁻ を2個渡す還元剤としてはたらきます。

⚗️ HNO₃（希）+ Cu → イオン反応式を作ろう
酸化剤：希HNO₃（e⁻ 3個） 還元剤：Cu（e⁻ 2個）
Step 0：半反応式を書く
酸化剤 HNO₃ + 3H⁺ + 3e⁻ → NO + 2H₂O
還元剤 Cu → Cu²⁺ + 2e⁻
Cu（単体）は Cu²⁺ になる還元剤（Cu：0→+2、e⁻ 2個渡す）。
e⁻ の最小公倍数：3と2 → 最小公倍数 = 6。酸化剤×2、還元剤×3。
Step 1：e⁻ の数を揃える（最小公倍数6）
🔢 最小公倍数で揃える
酸化剤のe⁻：3個　還元剤のe⁻：2個
最小公倍数 = 6
酸化剤の式 × 2　（3e⁻ × 2 = 6e⁻）
還元剤の式 × 3　（2e⁻ × 3 = 6e⁻）
酸化剤×2 2HNO₃ + 6H⁺ + 6e⁻ → 2NO + 4H₂O
還元剤×3 3Cu → 3Cu²⁺ + 6e⁻
Step 2：辺々加えて e⁻ を消去
＋ 辺々加える（e⁻ 消去）
2HNO₃ + 6H⁺ + 6e⁻ → 2NO + 4H₂O
3Cu → 3Cu²⁺ + 6e⁻
2HNO₃ + 6H⁺ + 3Cu → 2NO + 4H₂O + 3Cu²⁺
HNO₃ は左辺に、H⁺（硫酸由来）も左辺に。整理不要でこのまま完成。
完成！イオン反応式
2HNO₃ + 6H⁺ + 3Cu → 3Cu²⁺ + 2NO↑ + 4H₂O
Cu : 希HNO₃ = 3 : 2（ただしH⁺は別途必要）
  • NO は無色の気体。空気中で NO₂（赤褐色）に変化
  • 銅は希塩酸・希硫酸には溶けないが希硝酸・濃硝酸・熱濃硫酸には溶ける
  • 化学反応式：8HNO₃（希）+ 3Cu → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
✏️ 確認問題
問1希 HNO₃ と Cu の反応の係数合わせで、酸化剤と還元剤の倍数として正しいのはどれか。
正解はエ：酸化剤×2、還元剤×3。
HNO₃（e⁻ 3個）：Cu（e⁻ 2個）→ 最小公倍数 = 6
酸化剤 × 2（3×2=6）、還元剤 × 3（2×3=6）。
問2銅が溶けない酸はどれか。
正解はイ：希塩酸。
銅は希塩酸・希硫酸では溶けない（これらはイオン化傾向で Cu より H が大きいが、酸化剤としての力が弱い）。
希硝酸・濃硝酸・熱濃硫酸には溶けます（これらは酸化力を持つ）。

★ mol比の求め方

  • 例：KMnO₄（e⁻ 5個）vs FeSO₄（e⁻ 1個） → mol比 = 1:5（逆比）
  • e⁻ の総数が等しくなるように揃える → 最小公倍数を使う
