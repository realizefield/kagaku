Ch.14 ボルタ電池・ダニエル電池
⚡ §1 電池とは何か
🔋 まずここから！
電池とは、化学反応のエネルギーを電気エネルギーに変換する装置です。
鍵となるのは「酸化還元反応」です。
酸化（電子を失う反応）と還元（電子を得る反応）を2か所に分けて行うことで、電子が導線を通って流れ＝電流が生まれます。
🔑 まず「酸化」と「還元」を完全に理解しよう
酸化（される）
➡️ e⁻ を失う
電子を放出する反応
例：Zn → Zn²⁺ + 2e⁻
「e⁻ を出す側」
還元（される）
⬅️ e⁻ を得る
電子を受け取る反応
例：2H⁺ + 2e⁻ → H₂
「e⁻ を受け取る側」
💡 語呂合わせで覚えよう
「酸化 = 電子を失う（サンカ＝去る）」　「還元 = 電子を得る（カンゲン＝帰る）」
電子e⁻ が「去る」のが酸化、「帰ってくる」のが還元、と覚えましょう。
🔑 電池の正極・負極と酸化還元の関係
極 イオン化傾向 電子の動き 反応の種類
負極（－） 大きい方 e⁻ を放出 酸化される
正極（＋） 小さい方 e⁻ を受け取る 還元される
⚡ 電子は導線を通じて 負極 → 正極 へ流れます（電流の向きは逆）

🔋 §2 ボルタ電池
ボルタ電池：Zn板（負極）＋ Cu板（正極）＋ 希硫酸（H₂SO₄ aq）
表記：（－）Zn ｜ H₂SO₄ aq ｜ Cu（＋）
🔍 なぜ Zn が負極で Cu が正極なのか？
1
イオン化傾向を確認：Zn ＞ Cu
Znの方がイオン化傾向が大きい ＝ 陽イオンになりやすい
2
イオンになりやすい ＝ 電子を手放しやすい
→ Zn は電子を放出する側 → 酸化される → 負極
3
Cuはイオン化傾向が小さい → 電子を受け取る側 → 還元される → 正極
（正極ではCuが溶けるのではなく、H⁺が電子を受け取りH₂が発生）
📝 反応式を自分で書けるようにしよう（ボルタ電池）
【負極の反応式】
1
Zn がイオンになる：Zn → Zn²⁺
2
電荷を合わせる：左辺 0、右辺 +2 → 右辺に 2e⁻ を加える
負極（酸化）：Zn → Zn²⁺ + 2e⁻
✅ Znが電子を失っている → 酸化
【正極の反応式】
1
希硫酸中の H⁺ が電子を受け取りH₂になる：H⁺ → ½H₂
2
H⁺ が2個必要、電子も2個：2H⁺ + 2e⁻ → H₂
正極（還元）：2H⁺ + 2e⁻ → H₂↑
✅ H⁺が電子を受け取っている → 還元
⚠️ 分極（ぶんきょく）とは？
Cu板（正極）の表面に H₂ の泡が付着し、電子の移動を妨げるため、
しばらくすると電流が流れにくくなります。これを分極といいます。
→ ボルタ電池の大きな欠点。ダニエル電池はこれを解消しました。

🔋 §3 ダニエル電池
ダニエル電池：Zn板（負極）＋ Cu板（正極）＋ ZnSO₄ aq ／ CuSO₄ aq（素焼き板で仕切り）
表記：（－）Zn ｜ ZnSO₄ aq ｜｜ CuSO₄ aq ｜ Cu（＋）
🔍 なぜ酸化・還元になるのか（ダニエル電池）
負極（Zn）＝ 酸化
Zn ＞ Cu（イオン化傾向）
→ Zn は電子を手放しやすい
→ Zn が溶けて Zn²⁺ + 2e⁻
→ e⁻ を失う ＝ 酸化
正極（Cu）＝ 還元
CuSO₄ aq 中の Cu²⁺ がある
→ Cu²⁺ が e⁻ を受け取りやすい
→ Cu²⁺ + 2e⁻ → Cu
→ e⁻ を得る ＝ 還元
⚠️ ボルタとの違い：正極の反応が違う！
ボルタ：正極で H⁺ が還元 → H₂ 発生
ダニエル：正極で Cu²⁺ が還元 → Cu 析出（H₂は発生しない！）
📝 反応式を自分で書けるようにしよう（ダニエル電池）
【負極の反応式】
負極（酸化）：Zn → Zn²⁺ + 2e⁻
✅ ボルタ電池の負極と同じ。Znが電子を失う（酸化）。
【正極の反応式】
1
CuSO₄ aq の中に Cu²⁺ が溶けている
2
Cu²⁺ が導線から届いた電子を受け取り Cu に戻る
左辺 +2、右辺 0 → 左辺に 2e⁻ を加える
正極（還元）：Cu²⁺ + 2e⁻ → Cu↓（赤褐色析出）
✅ Cu²⁺が電子を受け取っている → 還元。Cu板に赤褐色のCuが析出。
2液が混ざると電池が働かなくなる！

もし ZnSO₄ と CuSO₄ が混ざると、
Zn + Cu²⁺ → Zn²⁺ + Cu　という反応が導線なしに直接起きてしまいます。
→ 電子が導線を流れなくなり、電流が取り出せません。

素焼き板の役割：
・小さな穴があり、イオン（SO₄²⁻, Zn²⁺）だけが通れる
・2液の直接混合を防ぎながら電荷バランスを保つ

ダニエル電池を長持ちさせるには？
・負極側（ZnSO₄）→ 薄くする（Znが溶けやすくなる）
・正極側（CuSO₄）→ 濃くする（Cu²⁺が多いほどCuが析出しやすい）
項目 ボルタ電池 ダニエル電池
電解液H₂SO₄ aq（1種）ZnSO₄ aq / CuSO₄ aq
負極反応Zn→Zn²⁺+2e⁻（酸化）Zn→Zn²⁺+2e⁻（酸化）
正極反応2H⁺+2e⁻→H₂↑（還元）Cu²⁺+2e⁻→Cu（還元）
正極の変化H₂ 気体が発生Cuが赤褐色で析出
分極あり（欠点）なし（改良点）
