練習問題：半反応式の組み合わせ

e⁻ の数を揃えてイオン反応式を作る手順を確認します。
mol比の計算・係数決定・共通項の消去を含む入試頻出の問題です。

1e⁻の数・係数・mol比の基本 基本
1
KMnO₄ が酸化剤としてはたらくとき、MnO₄⁻ 1mol が受け取る e⁻ は何 mol か。
正解はウ：5 mol。
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
Mn の酸化数：+7 → +2（5 減る → e⁻ 5個受け取る）。
2
FeSO₄ が還元剤としてはたらくとき、Fe²⁺ 1mol が渡す e⁻ は何 mol か。
正解はア：1 mol。
Fe²⁺ → Fe³⁺ + e⁻
Fe の酸化数：+2 → +3（1 増える → e⁻ 1個渡す）。多くの還元剤が2molなのに対し、FeSO₄ は1mol という点が特徴。
3
K₂Cr₂O₇ が酸化剤としてはたらくとき、Cr₂O₇²⁻ 1mol が受け取る e⁻ は何 mol か。
Cr₂O₇²⁻ → 2Cr³⁺　Cr が2個ある！
Cr の酸化数：+6 → +3（1個あたり 3 減る × Cr 2個 = ?）
正解はエ：6 mol。
Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
Cr：+6 → +3（3減る）× 2個 = 6e⁻。
4
KMnO₄ と FeSO₄ の mol 比（KMnO₄：FeSO₄）はいくらか。
KMnO₄（e⁻ 5個）と FeSO₄（e⁻ 1個）
e⁻ の最小公倍数 = ?　→ 酸化剤 × ?、還元剤 × ? mol比 = 係数の比
正解はイ：1：5。
e⁻ の最小公倍数 = 5。酸化剤×1（5e⁻）、還元剤×5（1e⁻×5=5e⁻）。
→ KMnO₄：FeSO₄ = 1：5。
5
KMnO₄ と (COOH)₂（シュウ酸）の mol 比（KMnO₄：(COOH)₂）はいくらか。
KMnO₄（e⁻ 5個）と (COOH)₂（e⁻ 2個）
e⁻ の最小公倍数 = 10　→ 酸化剤 × 2、還元剤 × 5
正解はウ：2：5。
e⁻ の最小公倍数 = 10。酸化剤×2（5×2=10e⁻）、還元剤×5（2×5=10e⁻）。
→ KMnO₄：(COOH)₂ = 2：5。
2イオン反応式の係数決定 標準
6
KMnO₄ と H₂O₂ のイオン反応式（下記）の係数 a, b, c の組み合わせとして正しいのはどれか。
a MnO₄⁻ + b H⁺ + 5H₂O₂ → a Mn²⁺ + c H₂O + 5O₂
KMnO₄（e⁻ 5個）と H₂O₂（e⁻ 2個）
最小公倍数 = 10 → 酸化剤×2、還元剤×5 酸化剤の式を2倍すると H₂O は何個？
正解はイ：a=2, b=6, c=8。
酸化剤×2：2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + 8H₂O
還元剤×5：5H₂O₂ → 5O₂ + 10H⁺ + 10e⁻
足すと：2MnO₄⁻ + 16H⁺ + 5H₂O₂ → 2Mn²⁺ + 8H₂O + 5O₂ + 10H⁺
H⁺ 整理：16 − 10 = 6H⁺ → a=2, b=6, c=8。
7
K₂Cr₂O₇ と FeSO₄ のイオン反応式として正しいのはどれか。
正解はア。
Cr₂O₇²⁻（e⁻ 6個）+ 6×Fe²⁺（e⁻ 1個×6）。
酸化剤：Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
還元剤×6：6Fe²⁺ → 6Fe³⁺ + 6e⁻　→ 足すとア。
8
希 HNO₃ と Cu のイオン反応式として正しいのはどれか。
希 HNO₃（e⁻ 3個）と Cu（e⁻ 2個）
最小公倍数 = 6 → 酸化剤×2、還元剤×3 酸化剤：2HNO₃ + 6H⁺ + 6e⁻ → 2NO + 4H₂O
正解はエ。
酸化剤×2：2HNO₃ + 6H⁺ + 6e⁻ → 2NO + 4H₂O
還元剤×3：3Cu → 3Cu²⁺ + 6e⁻
足すと：2HNO₃ + 6H⁺ + 3Cu → 3Cu²⁺ + 2NO + 4H₂O。
9
Cl₂ と FeSO₄ のイオン反応式として正しいのはどれか。
正解はウ。
酸化剤：Cl₂ + 2e⁻ → 2Cl⁻
還元剤×2：2Fe²⁺ → 2Fe³⁺ + 2e⁻
→ Cl₂ + 2Fe²⁺ → 2Cl⁻ + 2Fe³⁺（H₂O も H⁺ も不要）。
10
KMnO₄ とシュウ酸 (COOH)₂ のイオン反応式の H₂O の係数はいくつか。
酸化剤×2：2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + ? H₂O
MnO₄⁻ に O が 4個 → 4H₂O → 2倍すると？
正解はイ：8。
MnO₄⁻ の半反応式：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O（H₂O が4個）
酸化剤を2倍：2MnO₄⁻ + 16H⁺ + 10e⁻ → 2Mn²⁺ + 8H₂O。
3mol の計算（量的関係） 標準〜発展
11
KMnO₄ 0.1 mol と過不足なく反応する FeSO₄ は何 mol か。
KMnO₄ : FeSO₄ = 1 : 5（mol比）
KMnO₄ 0.1 mol × 5 = ?
正解はウ：0.5 mol。
KMnO₄：FeSO₄ = 1：5 より、KMnO₄ 0.1 mol に対して FeSO₄ = 0.1 × 5 = 0.5 mol。
12
KMnO₄ 0.02 mol と過不足なく反応する (COOH)₂ は何 mol か。
KMnO₄ : (COOH)₂ = 2 : 5（mol比）
KMnO₄ 0.02 mol → (COOH)₂ = 0.02 × 52 = ?
正解はア：0.05 mol。
KMnO₄：(COOH)₂ = 2：5 より、(COOH)₂ = 0.02 × 52 = 0.05 mol。
13
K₂Cr₂O₇ 0.1 mol と過不足なく反応する FeSO₄ は何 mol か。
正解はエ：0.6 mol。
K₂Cr₂O₇：FeSO₄ = 1：6 より、FeSO₄ = 0.1 × 6 = 0.6 mol。
14
KMnO₄ 水溶液（濃度 0.020 mol/L）10.0 mL が FeSO₄ 水溶液 25.0 mL と過不足なく反応した。FeSO₄ 水溶液の濃度は何 mol/L か。
KMnO₄：FeSO₄ = 1：5（mol比）
KMnO₄ の mol 数 = 0.020 × 10.01000 = 2.0×10⁻⁴ mol
FeSO₄ の mol 数 = KMnO₄ × 5 = ? FeSO₄ の濃度 = mol ÷ 体積(L)
正解はイ：0.040 mol/L。
KMnO₄：0.020 × 0.0100 = 2.0×10⁻⁴ mol
FeSO₄：2.0×10⁻⁴ × 5 = 1.0×10⁻³ mol
濃度：1.0×10⁻³ ÷ 0.0250 = 0.040 mol/L。
15
KMnO₄ 水溶液（0.010 mol/L）20.0 mL とシュウ酸水溶液 10.0 mL が過不足なく反応した。シュウ酸水溶液の濃度は何 mol/L か。
KMnO₄：(COOH)₂ = 2：5（mol比）
KMnO₄：0.010 × 0.0200 = 2.0×10⁻⁴ mol
(COOH)₂ = 2.0×10⁻⁴ × 52 = ?
正解はウ：0.050 mol/L。
KMnO₄：0.010 × 0.0200 = 2.0×10⁻⁴ mol
(COOH)₂：2.0×10⁻⁴ × 52 = 5.0×10⁻⁴ mol
濃度：5.0×10⁻⁴ ÷ 0.0100 = 0.050 mol/L。
4総合・応用問題 発展
16
イオン反応式 2MnO₄⁻ + 6H⁺ + 5H₂O₂ → 2Mn²⁺ + 8H₂O + 5O₂ について正しい記述はどれか。
正解はエ。
KMnO₄（MnO₄⁻）：Mn +7 → +2（減る → 還元される → 酸化剤）
H₂O₂：O −1 → 0（増える → 酸化される → 還元剤）→ O₂ を生成
H₂O₂ は通常酸化剤ですが、KMnO₄ のような強い酸化剤と反応するとき還元剤になります。
17
次の反応 Cu + 希 HNO₃ について誤っているのはどれか。
正解（誤り）はア。
銅は希塩酸・希硫酸には溶けません（イオン化傾向で Cu < H のため、これらの酸では反応しない）。
希硝酸・濃硝酸・熱濃硫酸など酸化力を持つ酸にのみ溶けます。
18
0.020 mol/L の KMnO₄ 水溶液 25.0 mL を用いてシュウ酸水溶液を滴定した。KMnO₄ の mol 数と反応した (COOH)₂ の mol 数をそれぞれ求めよ。
KMnO₄：(COOH)₂ = 2：5（mol比）
KMnO₄ の mol = 0.020 × 25.01000
(COOH)₂ = KMnO₄ の mol × 52
正解はウ。
KMnO₄：0.020 × 0.0250 = 5.0×10⁻⁴ mol
(COOH)₂：5.0×10⁻⁴ × 52 = 1.25×10⁻³ mol
19
次の半反応式 MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O と C₂O₄²⁻ → 2CO₂ + 2e⁻ を組み合わせてイオン反応式を作るとき、整数係数を最も小さくした場合の CO₂ の係数はいくつか。
e⁻ の最小公倍数 = 10 → 酸化剤×2、還元剤×5
5C₂O₄²⁻ → ? CO₂ + 10e⁻
正解はイ：10。
還元剤×5：5C₂O₄²⁻ → 10CO₂ + 10e⁻
C₂O₄²⁻ 1mol から CO₂ が2mol 生成 → 5倍すると CO₂ = 10mol。
20
次のうち、イオン反応式として誤っているものはどれか。
正解（誤り）はエ。
正しくは：2MnO₄⁻ + 16H⁺ + 5C₂O₄²⁻ → 2Mn²⁺ + 8H₂O + 10CO₂
エでは H₂O の係数が 10 になっていますが、正しくは 8 です。
酸化剤×2 → 8H₂O（MnO₄⁻ 1個で 4H₂O → 2倍で 8H₂O）。
