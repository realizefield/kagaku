Chapter 5 酸と塩基の定義 ― アレニウス ＆ ブレンステッド・ローリー
🤔 そもそも「酸」「塩基」って何？
レモンが酸っぱいのは「酸」だから。石けんがぬるぬるするのは「塩基性」だから。
でも「酸」「塩基」を化学的にどう定義するかは、実は何度もアップデートされてきました。
📜 定義の歴史
1884年　アレニウス（スウェーデン）が最初の定義を提唱
1923年　ブレンステッドとローリーがより広い定義を提唱
→ どちらも「正解」。場面によって使い分けます。
🍋 1. 酸の性質と H⁺
酸の正体は 水素イオン H⁺ です。これが酸の性質を示す原因です。
💧 水溶液中のH⁺の正体
H⁺ は実は非常に不安定なので、水 H₂O と結合してオキソニウムイオン H₃O⁺ になります。
H⁺ ＋ H₂O → H₃O⁺　（省略して H⁺ と表す）
酸の水溶液の性質：
性質説明
酸味があるレモン・酢など
リトマス紙青色 → 赤色 に変える
金属と反応水素ガスを発生する（例：Zn ＋ HCl）
🧼 2. 塩基の性質と OH⁻
塩基の正体は 水酸化物イオン OH⁻ です。
📌 塩基とアルカリの違い
・酸の性質を打ち消すものを 塩基 という
・塩基のうち、水に溶けやすいものを アルカリ という
　→ アルカリ ⊂ 塩基　（アルカリは塩基の一部）
塩基の水溶液の性質：
性質説明
苦味がある重曹（NaHCO₃）の苦み
リトマス紙赤色 → 青色 に変える
手がぬるぬる皮膚の脂肪を分解するため
【例】NaOH → Na⁺ ＋ OH⁻

🇸🇪 3. アレニウスの定義
📌 アレニウスの定義（1884年）
酸　＝ 水溶液中で電離して H⁺ を生じる物質
塩基 ＝ 水溶液中で電離して OH⁻ を生じる物質
具体的な電離式：
【酸】
HCl → H⁺ ＋ Cl⁻　（塩酸）
H₂SO₄ → 2H⁺ ＋ SO₄²⁻　（硫酸）
CH₃COOH ⇌ H⁺ ＋ CH₃COO⁻　（酢酸：弱酸なので ⇌）
【塩基】
NaOH → Na⁺ ＋ OH⁻　（水酸化ナトリウム）
Ca(OH)₂ → Ca²⁺ ＋ 2OH⁻　（水酸化カルシウム）
⚠️ アレニウスの定義の「限界」
🤔
困った問題
アンモニア NH₃ を水に溶かすと塩基性になる。
でも NH₃ の中に OH⁻ はない！
→ アレニウスの定義では説明できない…
🧐
実際の反応
NH₃ ＋ H₂O ⇌ NH₄⁺ ＋ OH⁻

確かに OH⁻ は出てくる！でも、NH₃ 自身には OH がない。なぜ？
💡 アレニウスの定義は「水溶液」だけに限定されており、気体や固体の反応を説明できません。そこでより広い定義が必要になりました。
🔬 4. ブレンステッド・ローリーの定義 ― なぜ必要？
📌 ブレンステッドとローリーが考えたこと（1923年）
「酸・塩基の本質は H⁺（水素イオン）のやり取り なのではないか？」
H⁺ を 渡すのが酸、H⁺ を 受け取るのが塩基。
これなら NH₃ も説明できるようになります。
「H⁺ が渡される」と言うけれど、なぜそんなことが起きるの？
ヒントは 電気陰性度（電子を引き寄せる力） にあります。

① H₂O の中の H は、なぜ「出やすい」のか？
H₂O では、O（電気陰性度 大）が H（電気陰性度 小）との結合電子を強く引き寄せています。
そのため H は「電子をほぼ持っていない」 δ⁺（部分的に正） の状態になっています。
→ H⁺ として抜けやすい状態になっています。
O δ⁻ ─── H δ⁺ 　→　H⁺ として離れやすい
② NH₃ が H⁺ を「引き寄せる」のはなぜ？
NH₃ の N 原子には 孤立電子対（非共有電子対） があります。
この電子対が、δ⁺ になっている H⁺ を引き寄せることができます。
→ これが「H⁺ を受け取る」＝ 塩基としてはたらく仕組みです。
N（孤立電子対 :）　＋　H δ⁺ 　→　N─H（結合）
🔑 まとめると…
・H は電気陰性度が小さく、O や N と結合すると電子を奪われて H⁺ として「出やすく」 なる
・N や O の 孤立電子対 が H⁺ を引き寄せる（受け取る）力を持つ
・だから H⁺ は「渡す側（酸）」から「受け取る側（塩基）」へ移動できるわけです。

酸化還元との共通点：電子の引き寄せ合い（電気陰性度）が反応を生む、という本質は同じです。
※ このレベルは大学化学の入り口です。入試では「H⁺のやりとり」を覚えれば十分です。
📌 ブレンステッド・ローリーの定義（重要）
酸　＝ H⁺ を 与える 物質
塩基 ＝ H⁺ を 受け取る 物質
💡 NH₃ が塩基になれる理由を見てみよう
反応式：NH₃ ＋ H₂O ⇌ NH₄⁺ ＋ OH⁻

① H₂O の中の H が「δ⁺」になっている理由
O は電気陰性度がとても大きいので、H−O 結合の電子を自分の方に引き寄せてしまいます。
その結果、H は電子を引っ張られて δ⁺（部分的に正の電荷） を帯びた状態になります。

δ⁺H ― δ⁻O ← 電子が O 側に偏っています
つまり H₂O の H は「引っ張られやすい（＝渡しやすい）状態」になっているわけです。
これが H₂O が H⁺ を渡す側 ＝ 酸 としてはたらける理由です。
② NH₃ が H⁺ を「受け取れる」理由
NH₃ の N には 孤立電子対（非共有電子対） があります。
この孤立電子対が、H⁺（陽子だけで電子を持っていない）を引き寄せることができます。

:NH₃ ← : が孤立電子対をあらわしています
H⁺ は陽子だけ（電子がない）なので、電子を持っている孤立電子対に引き寄せられます。
これが NH₃ が H⁺ を受け取る側 ＝ 塩基 としてはたらける理由です。

▶ 酸と塩基の性質

酸の性質を示す原因は H⁺（水素イオン） です。 酸の水溶液はリトマス紙を 青色から赤色 に変えます。 塩基の性質を示す原因は OH⁻（水酸化物イオン） です。 塩基の水溶液はリトマス紙を 赤色から青色 に変えます。

▶ アレニウスの定義

アレニウスの定義では、酸とは水溶液中で H⁺ を生じる物質、 塩基とは水溶液中で OH⁻ を生じる物質です。 この定義の限界は 水溶液以外 の反応が説明できないことです。 たとえば NH₃（アンモニア） は OH⁻ を持たないのに塩基性を示します。

▶ ブレンステッド・ローリーの定義

ブレンステッド・ローリーの定義では、酸とは H⁺ を 与える 物質、 塩基とは H⁺ を 受け取る 物質です。 NH₃ ＋ H₂O ⇌ NH₄⁺ ＋ OH⁻ において、NH₃ は H⁺ を 受け取る ので 塩基 であり、 H₂O は H⁺ を 与える ので 酸 としてはたらきます。 H₂O のように酸にも塩基にもなれる物質を 両性物質 といいます。

▶ 強酸・弱酸

電離度 α ≒ 1 の酸を 強酸、α ≪ 1 を 弱酸 といいます。 HCl・H₂SO₄・HNO₃ は 強酸、 CH₃COOH・H₂CO₃ は 弱酸 です。
