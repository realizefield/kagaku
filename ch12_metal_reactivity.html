Ch.12 金属の反応性
💧 §1 金属と水の反応
金属とイオン化傾向の関係で、水・酸・塩基との反応性が決まります。
まず「水との反応」をイオン化傾向の大きい順に整理します。
🔑 水との反応まとめ
・K・Ca・Na：常温の水と反応 → H₂ 発生
　　例：2Na + 2H₂O → 2NaOH + H₂↑
・Mg：熱水（沸騰した水）と反応 → H₂ 発生
　　例：Mg + 2H₂O → Mg(OH)₂ + H₂↑
・Al・Zn・Fe：高温の水蒸気と反応
・Ni・Sn・Pb・Cu 以下：水とは反応しない
💡 覚え方：K・Ca・Naは「常温でも水に反応するほど激しい」金属です。Naが水に触れると激しく反応するのを実験で見たことがある人もいますね。

🧪 §2 金属と酸の反応
希塩酸・希硫酸などの希酸、および酸化力のある硝酸・熱濃硫酸との反応を整理します。
⚠️ Pb（鉛）に注意！
PbはHよりイオン化傾向が大きいので希酸に溶けるはずですが…
・希塩酸 → 表面に PbCl₂（塩化鉛(II)） の白色被膜ができて反応停止
・希硫酸 → 表面に PbSO₄（硫酸鉛(II)） の白色被膜ができて反応停止
→ 実際には「溶けない」扱いになります！
酸化力のある酸（硝酸・熱濃硫酸）との反応は別ルールです：
🔑 Cu・Hg・Ag が溶ける酸
Cu・Hg・Agは希酸には溶けませんが、酸化力のある酸に溶けます：
・希硝酸：3Cu + 8HNO₃(希) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
・濃硝酸：Cu + 4HNO₃(濃) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
・熱濃硫酸：Cu + 2H₂SO₄(熱濃) → CuSO₄ + SO₂↑ + 2H₂O
🛡️ 不動態とは
Fe・Ni・Al は酸化力の強い酸（熱濃硫酸・濃硝酸）に浸すと、
表面に緻密な酸化被膜（不動態）ができ、内部まで反応が進まなくなります。

⚠️ よくあるミス：「Fe は酸に溶けない」は誤りです！
Fe は 希塩酸・希硫酸 などには溶けます（H₂ 発生）。
不動態になるのは 濃硝酸・熱濃硫酸 のみです。
🔑 王水（おうすい）
濃塩酸 : 濃硝酸 ＝ 3 : 1 の混合液を 王水 といいます。
王水には Pt（白金）・Au（金） も溶けます（他の酸には溶けません）。
→ 「金も溶かす王様の水」という意味です。
⚠️ テキストの修正
テキストには「濃硝酸：濃塩酸 ＝ 1：3」と記載されていますが、
正しくは 濃塩酸：濃硝酸 ＝ 3：1 です（塩酸が多い）。混同しないように注意！

⚗️ §2.5 両性金属（Al・Zn・Sn・Pb）
ほとんどの金属は酸に溶けて水素を発生しますが、塩基（NaOHなど）には溶けません。
ところが Al・Zn・Sn・Pb の4つは、酸にも塩基にも溶けるという特別な性質をもっています。
これらを 両性金属 といいます。

🔑 Alを例にした反応式
酸（希塩酸）との反応：
　2Al + 6HCl → 2AlCl₃ + 3H₂↑

塩基（NaOH水溶液）との反応：
　2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑
　（テトラヒドロキシドアルミン酸ナトリウムを生成しH₂発生）
🧠 ポイント：Al³⁺ イオンの性質

Alが酸にも塩基にも溶ける理由は、Al³⁺（アルミニウムイオン）の電荷が大きく、とても小さいイオンだからです。
小さくて電荷が大きいイオンは、周りの水分子や OH⁻ を強く引き寄せ、複雑な反応をします。

酸との反応のしくみ（希塩酸の例）
Al は電子を3個放出して Al³⁺ になります（酸化）。
溶液中の H⁺ がその電子を受け取って H₂↑ が発生します。
→ これはふつうの金属（Fe・Znなど）と同じしくみです。
塩基との反応のしくみ（NaOH水溶液の例）
Al(OH)₃（水酸化アルミニウム）は両性水酸化物です。
通常の水酸化物（例：Fe(OH)₃）は NaOH に溶けませんが、
Al(OH)₃ は OH⁻ をさらに受け取り [Al(OH)₄]⁻（錯イオン）になって溶けます。

これを酸塩基両性といい、Al³⁺ のイオン半径が非常に小さく電荷が大きいため、
OH⁻ を強く引きつける力があることが理由です。
🔑 Zn・Sn・Pb も同じしくみ
Zn：Zn + 2NaOH → Na₂ZnO₂ + H₂↑
    Zn(OH)₂ + 2OH⁻ → [Zn(OH)₄]²⁻（錯イオン）として溶ける

Sn・Pb も同様に両性を示しますが、高校入試では Al・Zn が特に重要です。
💡 まとめ：なぜ両性なのか（一言で）
Al³⁺・Zn²⁺ などはイオン半径が小さく電荷が大きいため、
水溶液中で OH⁻ と複雑な錯イオンを作れます。
→ 酸の中では「金属として H⁺ に電子を渡す」
→ 塩基の中では「OH⁻ を引き寄せて錯イオンになって溶ける」
この二重の能力が「両性」の正体です。

🧪 ミニテスト §3
Q6. Pt（白金）と Au（金）を溶かすことができる液体はどれですか？
Q7. 鉄（Fe）を濃硝酸に入れたとき起こることとして正しいのはどれですか？
Fe・Ni・Al を濃硝酸や熱濃硫酸に入れると、表面に Fe₂O₃・NiO・Al₂O₃ などの
緻密な酸化物被膜が生成します。この被膜は非常に薄いですが緻密で、
酸や酸素が内部に到達するのを防ぎます。

日常例：アルミニウムのアルマイト加工（Al₂O₃被膜を人工的に厚くする）や、
ステンレス鋼（Fe + Cr）の錆びにくさも同じ原理です。
