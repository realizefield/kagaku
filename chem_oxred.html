化学基礎 / 酸化還元(1)

酸化還元(1) ★★☆

酸化数の定義と計算・酸化還元の定義・酸化剤と還元剤の見分け方を学びます。高校化学で最も重要な単元のひとつです。

1酸化数の決め方
📖 説明

酸化数とは、原子がどれだけ「酸化されているか」を数値で表したものです。
以下の5つのルールで決まります。

ルール①
単体中の原子の酸化数は 0
例：Cl₂, O₂, Fe → すべて 0
ルール②
単原子イオンの酸化数はイオンの電荷と同じ
例：Cu²⁺ → +2、Cl⁻ → −1
ルール③
化合物中の H は +1、O は −2
（例外：NaH の H は −1、H₂O₂ の O は −1）
原則としてH=+1, O=−2 を使う
ルール④
化合物中の原子の酸化数の 総和 = 0
例：CuO → Cu(+2) + O(−2) = 0 ✓
ルール⑤
多原子イオン中の原子の酸化数の合計 = イオンの電荷
例：SO₄²⁻ → S(+6) + O(−2)×4 = −2 ✓　 → Sの酸化数 = +6

▶ 例題：KMnO₄ の Mn の酸化数

Kの酸化数 = +1（ルール②より）、O = −2 なので

(+1) + x + (−2)×4 = 0 　→　 x = +7

★ 覚えよう！
  • 単体の酸化数 = 0（H₂, O₂, Fe, Cu など）
  • 化合物中：H = +1（例外あり）、O = −2（例外あり）
  • 化合物の酸化数の総和 = 0、イオンなら = 電荷
  • 求める元素の酸化数を x とおいて計算する
✏️ ミニテスト（全4問）
問1単体の Cl₂ 中の Cl の酸化数はいくらか。
正解はイ：0。単体の酸化数はルール①より必ず 0 です。Cl₂ は単体(同じ原子が結合)なので Cl の酸化数 = 0。
問2NH₃ 中の N の酸化数はいくらか。H = +1 とする。
正解はウ：−3。N の酸化数を x とすると、x + (+1)×3 = 0 → x = −3。
問3HNO₃ 中の N の酸化数はいくらか。H = +1, O = −2 とする。
正解はエ：+5。x を N の酸化数とすると、(+1) + x + (−2)×3 = 0 → x = +5。
問4SO₄²⁻ 中の S の酸化数はいくらか。O = −2 とする。
正解はア：+6。多原子イオンなので x + (−2)×4 = −2 → x = +6。
2酸化還元の定義
📖 説明

「酸化」と「還元」は4つの観点から定義できます。酸化数の変化が最も重要です。

酸化数電子を水素を酸素を
酸化増えるはなすはなすもらう
還元減るもらうもらうはなす
Zn

Cu²⁺
+2 → 0

▲ CuSO₄ 水溶液に亜鉛板を浸す例（Zn＋Cu²⁺→Zn²⁺＋Cu）

▶ 重要ポイント

酸化と還元は必ず同時に起こります。ある物質が酸化されるとき、別の物質が必ず還元されています（電子の授受が起こるから）。

▶ 例題：H₂S ＋ O₂ → S ＋ H₂O

H₂S 中の S：−2 → 単体 S：0 ＝ 酸化数が増えた → 酸化

O₂ 中の O：0 → H₂O 中の O：−2 ＝ 酸化数が減った → 還元

★ 覚えよう！
  • 酸化数が増える → 酸化された
  • 酸化数が減る → 還元された
  • 酸化と還元は同時に起こる（酸化還元反応）
✏️ ミニテスト（全3問）
問1酸化とは、酸化数が増える変化のことである。
○ 正解です。酸化数が増える = 酸化、酸化数が減る = 還元。これが最も基本的な定義です。
問2酸化と還元は、常に同時に起こる。
○ 正解。電子は「渡す側」と「受け取る側」が必ずセットで存在します。片方だけの酸化・還元はありません。
問3次の反応 2Mg + O₂ → 2MgO で、酸化された物質はどれか。
正解はウ：Mg。Mg は単体(酸化数0)→ MgO 中(酸化数+2)と酸化数が増えたので「酸化された」。O₂ は 0 → −2 と減ったので「還元された」。
? なぜ「水素を離すと酸化」なのか？ ― 図解で納得

H は電気陰性度が 小さい（2.2）、S は 2.6 と大きい。
→ 結合中の電子は常に S 側に偏っており、H は「+1 の状態」で S にくっついています。

水素は電子を道連れにする
→ 残った S から電子が奪われる → S の酸化数が増える（−2 → 0）→ 酸化！
H（+1）が来ると相手は相対的にマイナスに引っ張られる
→ N の酸化数が減る（0 → −3）→ 還元！

H₂S → S + 2H⁺ + 2e⁻		 電子を道連れに H が去る

H が来れば +1 が増えて相手はマイナスに → 還元。H が去れば +1 が減って相手はプラスに → 酸化。
3酸化剤と還元剤
📖 説明
名称はたらき自身はどうなる酸化数
酸化剤 相手を酸化する 自身は還元される 減る
還元剤 相手を還元する 自身は酸化される 増える

▶ 重要な酸化剤・還元剤の一覧

代表的な酸化剤代表的な還元剤
KMnO₄（過マンガン酸カリウム）H₂（水素）
K₂Cr₂O₇（ニクロム酸カリウム）H₂S（硫化水素）
H₂O₂（過酸化水素）SO₂（二酸化硫黄）
Cl₂（塩素）Na, Mg, Zn, Fe（金属単体）
HNO₃（硝酸）シュウ酸 (COOH)₂
H₂SO₄（濃硫酸）H₂O₂（二面性：還元剤にもなる）
⚠️ 注意！ H₂O₂ の二面性
  • 通常：酸化剤（過酸化水素が電子を受け取る）
  • KMnO₄ と反応するとき：還元剤（KMnO₄ の方が強力な酸化剤のため）

▶ 例題：H₂O₂ ＋ 2KI ＋ 2HCl → 2H₂O ＋ I₂ ＋ 2KCl

H₂O₂ 中の O：−1 → H₂O 中の O：−2（減った → 還元された）
→ H₂O₂ は相手（KI）を酸化している → 酸化剤

KI 中の I：−1 → I₂ 中の I：0（増えた → 酸化された）
→ KI は相手（H₂O₂）を還元している → 還元剤

★ 覚え方のコツ
  • 酸化剤 → 自身が「還元される」（酸化数が減る）→ 電子をもらう側
  • 還元剤 → 自身が「酸化される」（酸化数が増える）→ 電子を渡す側
  • 名前と逆の変化をすると覚える！「酸化剤」→ 自身は「還元」される
✏️ ミニテスト（全4問）
問1酸化剤とはどのような物質か。
正解はイ。酸化剤 = 相手を酸化する → 自身は還元される（酸化数が減る）。
問22Al + 6HCl → 2AlCl₃ + 3H₂ の反応で、酸化剤はどれか。
正解はウ：HCl。HCl 中の H：+1 → H₂ 中の H：0（酸化数が減った→還元された）。相手（Al）を酸化しているので HCl が酸化剤。Alは +0→+3 と酸化された（還元剤）。
問3H₂O₂（過酸化水素）は常に酸化剤としてはたらく。
× 不正解。H₂O₂ は主に酸化剤ですが、KMnO₄（過マンガン酸カリウム）と反応するときは還元剤になります。このような物質を「二面性を持つ」といいます。
問4Pb + PbO₂ + 2H₂SO₄ → 2PbSO₄ + 2H₂O の反応で、還元剤はどれか。
正解はエ：Pb。Pb（単体）：0 → PbSO₄ 中の Pb：+2（酸化数が増えた → 酸化された = 還元剤）。PbO₂ 中の Pb：+4 → +2（減った = 酸化剤）。鉛蓄電池の反応です。
4半反応式の作り方（イオン反応式の基礎）
📖 説明

酸化剤・還元剤の反応は「半反応式」で表します。これを組み合わせてイオン反応式を作ります。

▶ 半反応式の作り方（手順）

① 主な変化を書く：反応物と生成物を書く

② O の数を合わせる：H₂O を加えて酸素を補う

③ H の数を合わせる：H⁺ を加えて水素を補う

④ 電荷を合わせる：e⁻（電子）を加えて電荷を合わせる

▶ 例：KMnO₄（酸化剤）の半反応式（酸性条件）

MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
Mn：+7 → +2（酸化数が5減る → 電子5個を受け取る）

▶ 例：H₂O₂（還元剤）の半反応式

H₂O₂ → O₂ + 2H⁺ + 2e⁻
O：−1 → 0（酸化数が増える → 電子2個を渡す）

▶ イオン反応式の作り方

酸化剤の半反応式と還元剤の半反応式を e⁻ の数が合うよう整数倍してから足し合わせます。

（例）MnO₄⁻ の e⁻ 5個 と H₂O₂ の e⁻ 2個 → 最小公倍数 10 に合わせる → 上式×2 + 下式×5

★ 覚えよう！
  • O が足りない → H₂O を加える（右辺に）
  • H が足りない → H⁺ を加える（左辺に、酸性条件）
  • 電荷を合わせる → e⁻ を加える
  • 組み合わせるときは e⁻ の数を最小公倍数に揃える
✏️ ミニテスト（全3問）
問1KMnO₄（酸性条件）の半反応式として正しいものはどれか。
正解はイ。手順通りに：①MnO₄⁻→Mn²⁺ ②右辺に4H₂O ③左辺に8H⁺ ④左辺に5e⁻（電荷を揃える）。
問2SO₂（還元剤）の半反応式として正しいものはどれか。
正解はウ。SO₂ の S：+4 → SO₄²⁻ の S：+6（酸化数+2増）。O を補うために左辺に 2H₂O。H を揃えるために右辺に 4H⁺。電荷合わせで右辺に 2e⁻。
問3酸化剤の半反応式と還元剤の半反応式を組み合わせるとき、何を合わせてから足すか。
正解はア：e⁻ の数。酸化剤が受け取る電子数 = 還元剤が渡す電子数 になるよう整数倍して合わせます。
