1酸化剤の反応前後・生成物 基本〜標準
1
KMnO₄ が硫酸酸性下で酸化剤としてはたらくとき、MnO₄⁻ の色と生成物の色の変化として正しいのはどれか。
正解はイ。
MnO₄⁻（赤紫色） + e⁻ → Mn²⁺（淡桃〜ほぼ無色）
Mn の酸化数：+7 → +2（5減る → e⁻ 5個受け取る）。
この色変化（赤紫色が消える）が滴定の終点の目印です。
2
K₂Cr₂O₇ が酸化剤としてはたらいた後の生成物と色はどれか。
正解はウ：Cr³⁺（緑色）。
Cr₂O₇²⁻（橙赤色） → Cr³⁺（緑色）
Cr の酸化数：+6 → +3（1個あたり3減る × 2個 = e⁻ 6個）。
※CrO₄²⁻（黄色）は塩基性条件で安定なイオンです（酸化剤の生成物ではない）。
3
H₂O₂ が酸性条件で酸化剤としてはたらくとき、生成するのは何か。
正解はア：H₂O。
酸性条件：H₂O₂ + 2H⁺ + 2e⁻ → 2H₂O
H⁺ が豊富にあるので OH⁻ が H⁺ と結合して H₂O まで進めます。
中性・塩基性では H⁺ が不足するため OH⁻ が生成します。
4
希硝酸と濃硝酸が酸化剤としてはたらくとき、それぞれ生成する気体の色はどれか。
正解はエ。
希 HNO₃ → NO（無色）（N：+5→+2、e⁻ 3個）
濃 HNO₃ → NO₂（赤褐色）（N：+5→+4、e⁻ 1個）
「希→NO、濃→NO₂」のセットは入試頻出です。
5
熱濃硫酸が酸化剤としてはたらくとき生成する気体はどれか。また、その S の酸化数変化は？
正解はイ：SO₂（S：+6→+4）。
熱濃 H₂SO₄ + 2H⁺ + 2e⁻ → SO₂ + 2H₂O
S の酸化数：+6 → +4（2減る → e⁻ 2個受け取る）。
⚠️ 希硫酸は酸化剤になりません（酸として H⁺ を放出するだけ）。
6
Cl₂ が酸化剤としてはたらくとき、1mol の Cl₂ が受け取る e⁻ の mol 数と生成物はどれか。
正解はウ。
Cl₂ + 2e⁻ → 2Cl⁻
Cl が 2個あるので e⁻ を合計 2mol 受け取ります（1個あたり1減る × 2個）。
色変化：黄緑色（Cl₂） → 無色（Cl⁻）
2還元剤の反応前後・生成物 基本〜標準
7
H₂S が還元剤としてはたらくとき、生成する物質と色はどれか。
正解はイ：S（黄色の固体）。
H₂S → S + 2H⁺ + 2e⁻
S の酸化数：−2 → 0（2増える → e⁻ 2個渡す）。
硫黄（S）は黄色の固体として析出します。反応後に溶液が白濁する。
8
SO₂ が還元剤としてはたらくとき、生成するイオンはどれか。また S の酸化数変化は？
正解はエ：SO₄²⁻、S：+4→+6。
SO₂ + 2H₂O → SO₄²⁻ + 4H⁺ + 2e⁻
S の酸化数：+4 → +6（2増える → e⁻ 2個渡す → 還元剤）。
⚠️ SO₂ が酸化剤のとき（H₂S との反応）は → S（単体）が生成します（区別！）。
9
シュウ酸 (COOH)₂ が還元剤としてはたらくとき、生成するのは何か。
正解はア：2CO₂（C：+3→+4）。
C₂O₄²⁻ → 2CO₂ + 2e⁻
C の酸化数：+3 → +4（1増える × 2個 = e⁻ 2個渡す）。
KMnO₄ 滴定の標定（濃度決定）に使われます。
10
KI（ヨウ化カリウム）が還元剤としてはたらくとき、生成する物質と色、デンプンとの反応はどれか。
正解はウ。
2I⁻ → I₂（褐色） + 2e⁻
生成した I₂ はデンプンと反応すると青紫色になります。
この反応がヨウ素滴定の終点判定に利用されます。
11
FeSO₄（Fe²⁺）が還元剤としてはたらくとき、1mol あたり渡す e⁻ の mol 数と生成物はどれか。
正解はイ：e⁻ 1mol、Fe³⁺。
Fe²⁺ → Fe³⁺ + e⁻
Fe の酸化数：+2 → +3（1増える → e⁻ 1個渡す）。
多くの還元剤が e⁻ 2mol を渡す中、FeSO₄ は e⁻ 1mol という点が特徴です。
3二面性と条件による違い 標準〜発展
12
H₂O₂ が還元剤としてはたらくとき、正しい記述はどれか。
正解はエ。
H₂O₂ は通常酸化剤（→ H₂O または OH⁻）ですが、
KMnO₄ のような超強力な酸化剤と出会うときだけ還元剤（→ O₂）になります。
KI との反応では H₂O₂ が酸化剤として I⁻ を I₂ に酸化します。
13
SO₂ が酸化剤としてはたらくとき、反応の相手と生成物の正しい組み合わせはどれか。
正解はア。
SO₂ が酸化剤になるのは H₂S のような強い還元剤と反応するときだけ。
2H₂S + SO₂ → 3S + 2H₂O（S の酸化数：+4 → 0）
イ・ウ・エは SO₂ が還元剤としてはたらく反応（SO₂ → SO₄²⁻）です。
14
H₂O₂ が中性・塩基性条件で酸化剤としてはたらくとき、生成するのはどれか。またその理由は？
中性・塩基性 = H⁺ がほとんどない
O が e⁻ を受け取って −2 になっても、H⁺ が来ないと…？
正解はウ：OH⁻。
H₂O₂ + 2e⁻ → 2OH⁻
中性・塩基性では H⁺ が不足するため、O²⁻ が H⁺ を拾えずOH⁻ のまま止まる。
酸性なら H⁺ が豊富なので OH⁻ + H⁺ → H₂O まで進めます。
15
SO₂ と H₂O₂ の二面性の理由として正しいものはどれか。
正解はイ。
SO₂ の S：+4（−2〜+6 の中間）→ 上にも下にも動ける
H₂O₂ の O：−1（−2〜0 の中間）→ 上にも下にも動ける
中間値の酸化数を持つから、相手次第で酸化剤にも還元剤にもなれます。
4総合問題・半反応式の確認 発展
16
次の反応で酸化剤と還元剤を正しく示したのはどれか。
H₂O₂ + 2KI + 2HCl → 2H₂O + I₂ + 2KCl
H₂O₂ の O と KI の I の酸化数変化を確認しよう
H₂O₂ の O：−1 → H₂O の O：?　　KI の I：−1 → I₂ の I：?
正解はエ。
H₂O₂ の O：−1 → H₂O の O：−2（減った → 還元された = 酸化剤）
KI の I：−1 → I₂ の I：0（増えた → 酸化された = 還元剤）
HCl は H⁺ と Cl⁻ を供給するだけで酸化数変化なし。
17
KMnO₄（酸性）の半反応式で正しいものはどれか。
正解はア。
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
O が 4個 → H₂O が 4個 → H が 8個 → H⁺ が 8個。
Mn：+7→+2（5減る → e⁻ 5個）。手順通りに確認しましょう。
18
H₂S の半反応式で正しいものはどれか。
正解はウ。
H₂S → S + 2H⁺ + 2e⁻
S の酸化数：−2 → 0（2増える → 還元剤、e⁻ を右辺に）。
O がないので H₂O 追加不要。H⁺ を右辺に 2個追加して完成。
19
次の酸化還元反応で、酸化剤が受け取った e⁻ の mol 数と還元剤が渡した e⁻ の mol 数について正しいのはどれか。
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O　（酸化剤の半反応式）
Fe²⁺ → Fe³⁺ + e⁻　（還元剤の半反応式）
正解はイ。
MnO₄⁻ 1mol が受け取る e⁻ = 5mol
Fe²⁺ 1mol が渡す e⁻ = 1mol
受け取る = 渡す にするには Fe²⁺ を 5mol 使う。
→ イオン反応式：MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 5Fe³⁺ + 4H₂O
20
次の反応の下線部の物質について正しい記述はどれか。
2SO₂ + O₂ → 2SO₃
（接触法による SO₃ の製造）
SO₂ の S の酸化数と SO₃ の S の酸化数を比べよう
SO₂：S = ?（O=−2）　SO₃：S = ?（O=−2）
正解はエ。
SO₂ の S：+4（x + (−2)×2 = 0 → x = +4）
SO₃ の S：+6（x + (−2)×3 = 0 → x = +6）
S の酸化数が +4 → +6 に増えた → 酸化された = 還元剤。
O₂（0 → −2）が還元された = 酸化剤。接触法は工業的な硫酸製造の重要反応。
