酸化・還元の定義

「酸化」「還元」は4つの視点から定義できます。
これらはすべて同じ現象を別の角度から見たものです。
まず具体的な反応を見て、それぞれの定義を感覚でつかみましょう。

🔰 初学者向け　丁寧解説
0まず「イメージ」からつかもう

「酸化」「還元」という言葉、聞いたことはありますか？
たとえばこんな身近な例があります。

🔩 鉄がさびる → 鉄が空気中の酸素と結びつく → 酸化
🏭 鉄を作る → 鉄鉱石から酸素を取り除く → 還元

これが「酸化・還元」の最もシンプルなイメージです。
しかし高校化学では、酸素が関係しない反応でも「酸化」「還元」が起こります。
そこで 4つの視点 から定義を整理していきます。

酸化（さんか）
物質が
「より酸化された状態」
に変わること
還元（かんげん）
物質が
「より還元された状態」
に変わること
★ 最重要ポイント
  • 酸化と還元は必ず同時に起こる（片方だけ起きることはない）
  • ある物質が酸化されるとき、必ず別の物質が還元されている
  • この2つが同時に起こる反応を 酸化還元反応 という
1視点①：酸素による定義

最もわかりやすい定義です。中学理科でも登場します。

酸化
酸素と結びつく
（酸素を得る）
還元
酸素を失う
（酸素を奪われる）
📝 具体例：酸化銅と水素の反応
CuO　+　H₂　→　Cu　+　H₂O
CuO：O を失った → 還元された
H₂：O と結びついた → 酸化された
📝 具体例：マグネシウムの燃焼
2Mg　+　O₂　→　2MgO
Mg：O と結びついた → 酸化された
O₂：Mg に O を与えた → （相手を酸化した）
⚠️ 注意
酸素による定義は「酸素が関係する反応」だけに使えます。
酸素が登場しない反応（例：Na が水に溶ける）には使えません。
→ そこで視点②③④が必要になります。
✏️ 確認問題（全3問）
問1 次の反応で酸化されたのはどれか。
2Cu + O₂ → 2CuO
正解はイ：Cu。
Cu は O₂ と結びついて CuO になった → 酸素を得た → 酸化された。
O₂ は Cu に酸素を与えた側（相手を酸化させた）。
問2 次の反応で還元されたのはどれか。
Fe₂O₃ + 3CO → 2Fe + 3CO₂
正解はウ：Fe₂O₃。
Fe₂O₃ は O を失って Fe になった → 酸素を失った → 還元された。
CO は O を得て CO₂ になった → 酸化された。
問3 酸化と還元について正しいのはどれか。
正解はア。
ある物質が O を得る（酸化）とき、必ず別の物質が O を失う（還元）。
酸素のやり取りは必ずペアで起きます。エは誤りで、酸素が関係しなくても酸化還元反応は起こります。
2視点②：水素による定義

有機化学や生化学でよく使われる定義です。酸素とは逆の対応になるので注意しましょう。

酸化
水素を失う
（水素を奪われる）
還元
水素と結びつく
（水素を得る）
⚠️ 酸素と逆！
酸素：得ると酸化・失うと還元
水素：失うと酸化・得ると還元（逆になっている）
これは「H の酸化数が +1 だから」です。→ 詳しくは下の「なぜ？」ボタンで確認。
📝 具体例：H₂S が酸化される
H₂S　→　S　+　2H⁺　+　2e⁻
H₂S：H を失った → 酸化された
📝 具体例：アンモニアの合成
N₂　+　3H₂　→　2NH₃
N₂：H と結びついた → 還元された
H₂：H を N₂ に与えた → 酸化された
✏️ 確認問題（全3問）
問1 水素による定義で、「還元された」のはどちらか。
CH₄ + 2O₂ → CO₂ + 2H₂O
正解はエ。
CH₄ は H を失った → 酸化された（視点②）。O₂ は H を得ていない。
この反応は「酸素による定義」で見る方が自然です（CH₄ が O₂ と結びついた）。
視点②は「H のやり取り」が中心の反応で使います。
問2 次のうち、水素による定義で酸化されているのはどれか。
正解はイ：HI。
HI → H₂ + I₂ で HI は H を「失った」（H が引き抜かれて I₂ になった）。
水素を失った → 酸化された（視点②）。
ウの N₂ は H を得た（還元）。ウとイを区別しましょう。
問3 酸素・水素の定義について正しいものはどれか。
正解はウ。
酸素：得ると→酸化、失うと→還元
水素：得ると→還元、失うと→酸化
酸素と水素で「得る・失う」の対応が逆になっている点が重要です。
3視点③：電子による定義

最も本質的な定義です。酸素も水素も関係しない反応でも使えます。
酸化還元の「本当の正体」は電子のやり取りです。

酸化
電子を失う
（電子を奪われる）
還元
電子を得る
（電子を受け取る）
📝 具体例：Zn と Cu²⁺ の反応（電子のやり取りがよくわかる）
Zn　+　Cu²⁺　→　Zn²⁺　+　Cu
Zn：e⁻ 2個を失った → 酸化された
Cu²⁺：e⁻ 2個を得た → 還元された
📝 半反応式で書くと…
酸化： Zn → Zn²⁺ + 2e⁻ （e⁻ を渡す）
還元： Cu²⁺ + 2e⁻ → Cu （e⁻ を受け取る）
★ 電子の定義が「最も本質的」な理由
  • 酸素・水素が関係しない反応にも使える
  • 視点①②③はすべて「電子のやり取り」に帰着する
  • 渡した e⁻ の数 = 受け取った e⁻ の数（電子は必ずペアで移動）
✏️ 確認問題（全3問）
問1 電子による定義で正しいものはどれか。
正解はア。
電子を失う→ 酸化（プラスになる方向）
電子を得る→ 還元（マイナスになる方向）
電子は必ず「渡す側」と「受け取る側」がセットです。
問2 反応 Fe + 2HCl → FeCl₂ + H₂ で酸化されたのはどれか。（電子の視点で考える）
正解はウ：Fe。
Fe（酸化数 0）→ Fe²⁺（酸化数 +2）：酸化数が増えた = 電子を失った = 酸化された。
HCl の H（+1）→ H₂（0）：酸化数が減った = 電子を得た = 還元された。
問3 酸化剤とはどのような物質か。（電子の視点で）
正解はエ。
酸化剤 = 相手から電子を受け取る（自分は還元される）→ 相手を酸化させる。
「酸化剤」という名前ですが、自分自身は「還元される」点に注意！
逆に「還元剤」は電子を渡して（自分は酸化されて）相手を還元させます。
4視点④：酸化数による定義（最も広い定義）

第1章で学んだ「酸化数」を使った定義です。これが最も広く・正確に使える定義です。
酸化数が変化したかどうかを見るだけで、酸化・還元の判定ができます。

酸化
酸化数が増える
還元
酸化数が減る
▼ 反応 Zn + Cu²⁺ → Zn²⁺ + Cu での酸化数の変化
Zn
0 → +2
酸化数が増えた
酸化された
⇄
Cu²⁺
+2 → 0
酸化数が減った
還元された
📝 2Mg + O₂ → 2MgO で確認
Mg：0 → +2（増加 → 酸化）
O：0 → −2（減少 → 還元）
★ 酸化数による定義が「最も広い」理由
  • 酸素・水素・電子どの変化も「酸化数の増減」として表現できる
  • 複雑な反応でも「酸化数が変化したか」だけで判定できる
  • 酸化数が変化しない反応 = 酸化還元反応ではない（例：中和、沈殿生成）
✏️ 確認問題（全3問）
問1 酸化数による定義で正しいものはどれか。
正解はイ。
酸化数が増える→ 酸化（電子を失った）
酸化数が減る→ 還元（電子を得た）
この対応は必ず暗記しましょう。
問2 次の反応のうち、酸化還元反応でないものはどれか。
正解はウ：NaCl + AgNO₃ → AgCl + NaNO₃。
ウは沈殿生成反応（AgCl ↓）。全元素の酸化数を確認すると Na(+1)、Cl(−1)、Ag(+1)、N(+5)、O(−2) — すべて変化なし。
ア：Na は 0 → +1（酸化）、H は +1 → 0（還元）。
イ：H は 0 → +1（酸化）、O は 0 → −2（還元）。
エ：Cl は 0 → −1（還元）、I は −1 → 0（酸化）。
問3 反応 H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O について正しいのはどれか。
正解はア。
H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O は酸塩基の中和反応です。
全元素の酸化数：H(+1→+1)、S(+6→+6)、O(−2→−2)、Na(+1→+1) — すべて変化なし。
酸化数が変化していない = 酸化還元反応ではありません。
54つの定義の総まとめ

4つの定義をまとめると下の表になります。
どの視点も「同じ現象」を別の角度から見ているだけです。

視点 酸化された 還元された
① 酸素 酸素を得た 酸素を失った
② 水素 水素を失った 水素を得た
③ 電子 電子を失った 電子を得た
④ 酸化数 酸化数が増えた 酸化数が減った
★ 最終まとめ
  • 酸化 ＝「電子を失う」＝「酸化数が増える」
  • 還元 ＝「電子を得る」＝「酸化数が減る」
  • 酸化と還元は必ず同時に起こる
  • 酸化数が変化しない反応は酸化還元反応ではない（中和、沈殿など）
✏️ 総合確認問題（全4問）
問1 次の記述のうち正しいものはどれか。
正解はエ：還元されると酸化数が減る。
ア：酸化 → 酸化数が増える（誤）
イ：還元 → 電子を得る（誤）
ウ：水素を得ると還元（誤）
エ：還元 = 酸化数が減る（○）
問2 反応 2Na + Cl₂ → 2NaCl で Na が「酸化された」ことを示す根拠として正しいのはどれか。
正解はイ。
Na（単体、酸化数 0）→ NaCl 中の Na（酸化数 +1）：酸化数が増えた → 酸化された。
酸化の根拠は「酸化数の増加」または「電子を失ったこと」で示します。
問3 次の4つの反応のうち、酸化還元反応はいくつあるか。
① 2H₂ + O₂ → 2H₂O
② HCl + NaOH → NaCl + H₂O
③ Zn + 2HCl → ZnCl₂ + H₂
④ BaCl₂ + Na₂SO₄ → BaSO₄↓ + 2NaCl
正解はウ：2つ（①と③）。
①：H(0→+1)酸化、O(0→−2)還元 ✓ 酸化還元
②：中和反応。全酸化数変化なし ✗
③：Zn(0→+2)酸化、H(+1→0)還元 ✓ 酸化還元
④：沈殿生成反応。全酸化数変化なし ✗
問4 酸化剤と還元剤について正しい記述はどれか。
正解はア。
酸化剤：相手から電子を受け取る → 自身の酸化数が減る → 自身は還元される
還元剤：相手に電子を渡す → 自身の酸化数が増える → 自身は酸化される
「酸化剤なのに自分は還元される」という逆の関係をしっかり覚えましょう。

❓ なぜ「水素を離すと酸化」になるのか

カギは「H の酸化数が +1」であること

H（水素）は化合物中で酸化数 +1 をとります。
つまり H は「プラスの電荷を帯びた状態」で他の原子にくっついています。

H が離れるとき ── 電気陰性度から考える

H の電気陰性度（電子を引き寄せる力）は 2.2 と小さく、
S（2.6）、N（3.0）、O（3.4）などより小さいため、
結合中の電子は常に相手側に偏っています。

H₂S の場合：S ──── H（電子は S 側に偏る）

H が離れるとき、H は電子を道連れにして去るため、
残された S の電子が減り → 酸化数が増える → 酸化された。

図でまとめると

水素の動き相手の酸化数結果
H を失う 増える 酸化
H を得る 減る 還元
一言まとめ：
「H の酸化数は +1 という1行に戻ればOK」
H が去れば +1 が出ていく → 相手の酸化数が増える → 酸化
H が来れば +1 がやってくる → 相手の酸化数が減る → 還元
