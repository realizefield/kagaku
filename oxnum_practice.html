1単体・単原子イオンの酸化数（ルール①②） 基本
📌 復習
ルール①：単体の酸化数 = 0（H₂, O₂, Fe, Cu など）
ルール②：単原子イオンの酸化数 = イオンの電荷（Na⁺ → +1, Cl⁻ → −1）
1
単体である Cu（銅）の酸化数はいくらか。
正解はイ：0。
Cu は銅の単体。ルール①より単体の酸化数 = 0。
「CuO の Cu = +2」と混同しないこと。CuO は化合物なので別のルールを使います。
2
Fe³⁺（鉄(III)イオン）の酸化数はいくらか。
正解はウ：+3。
Fe³⁺ は単原子イオン。ルール②よりイオンの電荷 = +3。
（単体 Fe の酸化数 = 0 とは区別しましょう。）
3
O₂（酸素の単体）中の O の酸化数はいくらか。
正解はア：0。
O₂ は酸素の単体。ルール①より 0。
「O の酸化数は −2 」はあくまで化合物中でのルール。単体には適用しません。
4
S²⁻（硫化物イオン）の酸化数はいくらか。
正解はエ：−2。
S²⁻ は単原子イオン（電荷 −2）。ルール②より −2。
単体 S₈（硫黄）は酸化数 0 です。
2化合物の酸化数を計算しよう（ルール③④） 標準
📌 復習
化合物中：H = +1、O = −2（例外あり）
化合物の酸化数の合計 = 0
求めたい元素を x とおいて方程式を立てる！
5
CO₂（二酸化炭素）中の C の酸化数はいくらか。
O = −2 とする
C の酸化数を x とおく
x + (−2) × 2 = 0 x − 4 = 0 → x = ?
正解はイ：+4。
x + (−2)×2 = 0 → x = +4。
CO₂ は燃焼や酸化の生成物として頻出。C の最大酸化数は +4 です。
6
NH₄⁺（アンモニウムイオン）中の N の酸化数はいくらか。
H = +1。NH₄⁺ の電荷は +1
NH₄⁺ は多原子イオン（電荷 +1）→ 合計 = +1
N の酸化数を x とおく
x + (+1) × 4 = +1 x + 4 = 1 → x = ?
正解はウ：−3。
NH₄⁺ は多原子イオン（電荷 +1）→ 合計 = +1。
x + (+1)×4 = +1 → x + 4 = 1 → x = −3。
NH₃ の N も −3 です（H が増えても N の酸化数は変わらない）。
7
HNO₃（硝酸）中の N の酸化数はいくらか。
H = +1、O = −2
H が 1個、O が 3個、N を x とおく
(+1) + x + (−2) × 3 = 0 1 + x − 6 = 0 → x = ?
正解はア：+5。
(+1) + x + (−2)×3 = 0 → x − 5 = 0 → x = +5。
HNO₃ は重要な酸化剤。N の酸化数 +5 は覚えておきましょう。
8
CuSO₄（硫酸銅）中の S の酸化数はいくらか。
Cu = +2、O = −2
CuSO₄ は化合物 → 合計 = 0
Cu = +2、O が 4個
(+2) + x + (−2) × 4 = 0 2 + x − 8 = 0 → x = ?
正解はエ：+6。
(+2) + x + (−2)×4 = 0 → 2 + x − 8 = 0 → x = +6。
SO₄²⁻ の中の S は常に +6 です（H₂SO₄ も同じ）。
3多原子イオンの酸化数を計算しよう（ルール⑤） 標準
📌 復習
多原子イオン中の酸化数の合計 = イオンの電荷
例：SO₄²⁻ なら合計 = −2、NH₄⁺ なら合計 = +1
9
MnO₄⁻（過マンガン酸イオン）中の Mn の酸化数はいくらか。
O = −2。このイオンの電荷は −1
★ レッスン例題3 の類題
多原子イオン（電荷 −1）→ 合計 = −1
Mn の酸化数を x とおく（O が 4個）
x + (−2) × 4 = −1 x − 8 = −1 → x = ?
正解はア：+7。
多原子イオン（電荷 −1）→ 合計 = −1。
x + (−2)×4 = −1 → x − 8 = −1 → x = +7。
KMnO₄ の Mn も同じ +7。酸化剤として反応すると Mn²⁺（+2）になります。
10
NO₂⁻（亜硝酸イオン）中の N の酸化数はいくらか。
O = −2。このイオンの電荷は −1
多原子イオン（電荷 −1）→ 合計 = −1
x + (−2) × 2 = −1 x − 4 = −1 → x = ?
正解はイ：+3。
x + (−2)×2 = −1 → x = +3。
NO₃⁻（硝酸イオン）の N = +5 と区別しましょう。
11
PO₄³⁻（リン酸イオン）中の P の酸化数はいくらか。
O = −2。このイオンの電荷は −3
多原子イオン（電荷 −3）→ 合計 = −3
x + (−2) × 4 = −3 x − 8 = −3 → x = ?
正解はウ：+5。
x + (−2)×4 = −3 → x = +5。
P の最大酸化数は +5 で、リン酸（H₃PO₄）の P も +5 です。
12
Cr₂O₇²⁻（ニクロム酸イオン）中の Cr の酸化数はいくらか。
O = −2。このイオンの電荷は −2
★ レッスン例題4 の類題（Cr が 2個！）
多原子イオン（電荷 −2）→ 合計 = −2
Cr が 2個 あるので 2x とおく（O が 7個）
2x + (−2) × 7 = −2 2x − 14 = −2 → 2x = 12 → x = ?
正解はイ：+6。
Cr が 2個 → 方程式は 2x + (−2)×7 = −2。
2x = 12 → x = +6。
ウの +12 は「2x = 12 の x」を 12 と読み間違えた典型的なミスです。
4例外・応用問題 発展
📌 例外の確認
H₂O₂ の O = −1（例外）／ NaH の H = −1（例外）／ F は常に −1
迷ったときは必ず x をおいて計算で確かめよう！
13
Na₂O₂（過酸化ナトリウム）中の O の酸化数はいくらか。
Na = +1
Na₂O₂ は化合物 → 合計 = 0
Na が 2個（+1×2 = +2）、O の酸化数を x（O が 2個）
(+1) × 2 + x × 2 = 0 2 + 2x = 0 → x = ?
正解はウ：−1。
(+1)×2 + 2x = 0 → 2x = −2 → x = −1。
Na₂O₂（過酸化ナトリウム）も H₂O₂ と同じく O−O 結合を持つため O = −1 になります。
14
CaH₂（水素化カルシウム）中の H の酸化数はいくらか。
Ca = +2
CaH₂ は化合物 → 合計 = 0
Ca = +2、H の酸化数を x（H が 2個）
(+2) + x × 2 = 0 2 + 2x = 0 → x = ?
正解はア：−1。
(+2) + 2x = 0 → x = −1。
金属（Ca）と H が結合するときは H が電子を受け取るので −1 になります（例外）。
15
S₂O₃²⁻（チオ硫酸イオン）中の S の酸化数はいくらか。
O = −2。このイオンの電荷は −2
⚠️ S が 2個あります
多原子イオン（電荷 −2）→ 合計 = −2
S が 2個 → 2x とおく（O が 3個）
2x + (−2) × 3 = −2 2x − 6 = −2 → 2x = 4 → x = ?
正解はエ：+2。
S が 2個 → 2x + (−2)×3 = −2 → 2x = 4 → x = +2。
チオ硫酸ナトリウム（Na₂S₂O₃）はヨウ素滴定の還元剤として重要です。
16
Fe₃O₄（四酸化三鉄）中の Fe の酸化数はいくらか。
O = −2。Fe が 3個あります
⚠️ 分数になる場合があります
Fe₃O₄ は化合物 → 合計 = 0
Fe が 3個 → 3x、O が 4個
3x + (−2) × 4 = 0 3x − 8 = 0 → 3x = 8 → x = ?
正解はイ：+83。
3x + (−2)×4 = 0 → 3x = 8 → x = +83（分数）。
Fe₃O₄ は Fe²⁺ と Fe³⁺ が混在した特殊な化合物（磁鉄鉱）なので、平均すると分数になります。
5重要化合物の酸化数 — 総まとめ 入試頻出
📌 入試で特に重要な物質
KMnO₄・K₂Cr₂O₇・HNO₃・H₂SO₄・H₂O₂・Na₂S₂O₃
これらの中心元素の酸化数はすぐに言えるようにしましょう。
17
KMnO₄（過マンガン酸カリウム）中の Mn の酸化数はいくらか。
K = +1、O = −2
化合物 → 合計 = 0。K = +1、O が 4個
(+1) + x + (−2) × 4 = 0 1 + x − 8 = 0 → x = ?
正解はア：+7。
(+1) + x + (−2)×4 = 0 → x = +7。
KMnO₄（赤紫色）は酸化剤。反応後は Mn²⁺（+2、ほぼ無色）になります。
18
K₂Cr₂O₇（ニクロム酸カリウム）中の Cr の酸化数はいくらか。
K = +1、O = −2
化合物 → 合計 = 0。K が 2個、Cr が 2個 → 2x、O が 7個
(+1) × 2 + 2x + (−2) × 7 = 0 2 + 2x − 14 = 0 → 2x = 12 → x = ?
正解はウ：+6。
(+1)×2 + 2x + (−2)×7 = 0 → 2x = 12 → x = +6。
K₂Cr₂O₇（橙色）は酸化剤。反応後は Cr³⁺（+3、緑色）になります。
19
H₂SO₃（亜硫酸）中の S の酸化数はいくらか。
H = +1、O = −2
化合物 → 合計 = 0。H が 2個、O が 3個
(+1) × 2 + x + (−2) × 3 = 0 2 + x − 6 = 0 → x = ?
正解はエ：+4。
(+1)×2 + x + (−2)×3 = 0 → x = +4。
H₂SO₃（S = +4）と H₂SO₄（S = +6）の違いを区別しましょう。SO₂ も S = +4 です。
20
次のうち、下線部の元素の酸化数が最も大きいものはどれか。
正解はイ：HNO₃ の N = +5。
各選択肢の N の酸化数：
ア：NH₃ → x + (+1)×3 = 0 → −3
イ：HNO₃ → (+1) + x + (−2)×3 = 0 → +5（最大）
ウ：N₂ は単体 → 0
エ：NO₂ → x + (−2)×2 = 0 → +4
