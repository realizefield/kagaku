練習問題：酸化・還元の定義

4つの視点（酸素・水素・電子・酸化数）をしっかり確認します。
わからないときは「ヒント」ボタンで手がかりを確認できます。

📝 全20問 💡 ヒント付き ⚗️ 4つの視点を総合
0 / 20 回答済み
1酸素・水素による定義（視点①②） 基本
1
次の反応で「酸化された」物質はどれか。
2Fe + 32 O₂ → Fe₂O₃
正解はイ：Fe。
Fe は O₂ と結びついた（酸素を得た） → 酸化された。
O₂ は Fe に酸素を与えた側（Fe を酸化させた酸化剤）。鉄が錆びる反応です。
2
次の反応で「還元された」物質はどれか。
CuO + H₂ → Cu + H₂O
正解はウ：CuO。
CuO は O を失って Cu になった（酸素を失った） → 還元された。
H₂ は O を得て H₂O になった → 酸化された。
3
水素による定義で「酸化された」のはどれか。
2HI → H₂ + I₂
正解はア：HI（I の部分）。
HI → I₂ で、I は H を失った → 水素を失った → 酸化された（視点②）。
H は HI から離れて H₂ になった → 水素を得た → 還元された。
4
酸素と水素の定義で「得る・失う」の対応が正しいものはどれか。
正解はエ。
酸素（O の酸化数は −2）：得ると酸化、失うと還元
水素（H の酸化数は +1）：得ると還元、失うと酸化
O と H では「得る」の対応が逆になります。
5
次の反応のうち、酸素による定義で「還元」が起きているのはどれか。
正解はイ。
Fe₂O₃ が O を失って Fe になった → 還元された（製鉄の反応）。
ア・ウ・エはすべて単体が O と結びついた（酸化）。
2電子・酸化数による定義（視点③④） 標準
6
電子による定義で正しいのはどれか。
正解はウ。
電子を失う → 酸化（酸化数が増える）
電子を得る → 還元（酸化数が減る）
電子のやり取りは酸化還元の「本質」です。
7
次の反応で酸化数が増えた原子はどれか。
Zn + 2HCl → ZnCl₂ + H₂
各元素の酸化数を求めよう
Zn：単体→？　H（HCl中）：+1→？（H₂中）
正解はア：Zn。
Zn：0（単体）→ ZnCl₂ 中 +2（増えた → 酸化された）
H：+1（HCl 中）→ 0（H₂ 単体）（減った → 還元された）
Cl は −1 のまま変化なし。
8
次の反応で「還元された」原子はどれか。
2Na + 2H₂O → 2NaOH + H₂
H₂O 中の H の酸化数は？　H₂（単体）の酸化数は？ H₂O：H = +1　→　H₂：H = ?
正解はエ。
H₂O の H：+1 → H₂（単体）の H：0（減った → 還元された）
Na：0（単体）→ NaOH 中 +1（増えた → 酸化された）
O は −2 のまま、NaOH の H（+1）も変化なし。
9
次の反応のうち、酸化還元反応でないものはどれか。
正解はイ。
CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂ は塩酸との反応（酸との反応）。
全元素（Ca、C、O、H、Cl）の酸化数が変化しない → 酸化還元反応ではない。
ア：Fe(0→+2)酸化・H(+1→0)還元　ウ：I(−1→0)酸化・Cl(0→−1)還元　エ：Mn(+4→+2)還元・Cl(−1→0)酸化
10
「酸化数が変化しない反応」の組み合わせはどれか。
正解はウ。
中和反応（HCl + NaOH → NaCl + H₂O）と沈殿生成反応（BaCl₂ + Na₂SO₄ → BaSO₄↓ + 2NaCl）は
いずれも酸化数が変化しない → 酸化還元反応ではない。
燃焼・電気分解はどちらも酸化数が変化する酸化還元反応。
3酸化剤・還元剤の識別 標準
11
酸化剤についての記述で正しいものはどれか。
正解はイ。
酸化剤 = 相手から電子を受け取る → 相手を酸化させる → 自分は還元される（酸化数が減る）。
「酸化剤なのに自分は還元される」という逆の関係が重要です。
12
次の反応で「還元剤」はどれか。
2Na + Cl₂ → 2NaCl
酸化数の変化を確認しよう
Na：0 → ?（NaCl中）　Cl：0 → ?（NaCl中） 還元剤は「電子を渡して自分が酸化される物質」
正解はア：Na。
Na：0 → +1（増えた → 酸化された = 還元剤）
Cl：0 → −1（減った → 還元された = 酸化剤）
「酸化された物質」が還元剤、「還元された物質」が酸化剤です。
13
次の反応で酸化剤はどれか。
H₂SO₄ + Zn → ZnSO₄ + H₂↑
H の酸化数変化に注目しよう
H₂SO₄ 中の H：+1 → H₂（単体）の H：? 酸化剤は「還元される（酸化数が減る）物質」
正解はエ：H₂SO₄ の H。
H₂SO₄ の H：+1 → H₂ の H：0（減った → 還元された = 酸化剤）
Zn：0 → +2（増えた → 酸化された = 還元剤）
S は +6 のまま変化なし（希硫酸は酸として反応し、S の酸化数は変わらない）。
14
次の反応で酸化剤と還元剤を正しく示したものはどれか。
Cl₂ + 2KI → 2KCl + I₂
正解はウ。
Cl：0（Cl₂）→ −1（KCl）（減った → 還元された = 酸化剤）
I：−1（KI）→ 0（I₂）（増えた → 酸化された = 還元剤）
生成した I₂ は褐色で、デンプンと反応すると青紫色になります。
15
次の反応で自己酸化還元（不均化）が起きているのはどれか。
正解はア（と エも自己酸化還元ですが選択肢の中でア）。
2H₂O₂ → 2H₂O + O₂ では、同じ H₂O₂ の O（−1）が
→ H₂O の O（−2）に還元された部分と → O₂（0）に酸化された部分に分かれる。
同一物質が酸化剤にも還元剤にもなる反応を自己酸化還元（不均化）といいます。
4総合問題 発展
16
次の反応を酸化数の変化で分析した。正しいものはどれか。
3NO₂ + H₂O → 2HNO₃ + NO
NO₂ の N、HNO₃ の N、NO の N それぞれの酸化数を求めよう
NO₂：N = ?　HNO₃：N = ?　NO：N = ?
正解はエ。
NO₂ の N：+4　→　HNO₃ の N：+5（増加 → 酸化された）
NO₂ の N：+4　→　NO の N：+2（減少 → 還元された）
同じ NO₂ が酸化剤にも還元剤にもなる自己酸化還元反応です。
17
次の4つの反応のうち、酸化還元反応はいくつか。
① 2Na + 2H₂O → 2NaOH + H₂
② AgNO₃ + NaCl → AgCl↓ + NaNO₃
③ 2KMnO₄ → K₂MnO₄ + MnO₂ + O₂
④ NH₃ + HCl → NH₄Cl
正解はイ：2つ（①と③）。
①：Na(0→+1)酸化・H(+1→0)還元 ✓
②：沈殿生成。全酸化数変化なし ✗
③：Mn(+7→+6)と(+7→+4)自己酸化還元・O(−2→0)酸化 ✓
④：酸塩基反応（中和）。全酸化数変化なし ✗
18
次の反応の下線部の変化として正しいものはどれか。
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
MnO₄⁻ 中の Mn の酸化数と、Mn²⁺ の酸化数を比べよう
MnO₄⁻：Mn = ?（O=−2、イオンの電荷=−1）
正解はウ。
MnO₄⁻：x + (−2)×4 = −1 → x = +7
Mn²⁺：+2
+7 → +2（5 減った → 還元された）→ MnO₄⁻ は酸化剤（e⁻ を 5個受け取る）。
19
次の記述のうち誤っているものはどれか。
正解（誤り）はア。
酸化と還元は必ず同時・同一反応内で起こります。
電子は「渡す側」と「受け取る側」が必ずセットで存在し、片方だけでは起こりません。
イ・ウ・エはすべて正しい記述です。
20
次の反応で下線部の酸化数変化と役割の組み合わせが正しいものはどれか。
2H₂O₂ + SO₂ → H₂SO₄
（H₂SO₄ 中：H=+1、S=+6、O=−2）
H₂O₂ の O 酸化数と H₂SO₄ の O 酸化数を比べよう
SO₂ の S 酸化数と H₂SO₄ の S 酸化数を比べよう
H₂O₂ の O：−1　SO₂ の S：+4
正解はエ。
H₂O₂ の O：−1 → H₂SO₄ の O：−2（減った → 還元された = 酸化剤）
SO₂ の S：+4 → H₂SO₄ の S：+6（増えた → 酸化された = 還元剤）
H₂O₂ が通常の役割（酸化剤）として SO₂（還元剤）を酸化しています。
