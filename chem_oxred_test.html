単元末テスト：酸化還元(1) ★★★

大学入試問題から構成した本番レベルの問題です。酸化数・酸化還元の識別・酸化剤・還元剤・滴定計算を総合的に確認します。

1大問1：酸化数の総合問題 〈大阪工業大・改〉
🏆 大阪工業大学 入試問題（改題）
ア〜ケに当てはまる語句・数値を答えよ。
一般に、酸化とは【ア】と化合する反応、または【イ】を失う反応を意味し、還元とは【イ】と化合する反応、または【ア】を失う反応を意味する。
原子やイオンが電子を失い【ウ】が増加すれば酸化された、電子を受け取り【ウ】が減少すれば還元されたという。
酸化還元反応において、【ク】は電子を受け取り還元される物質であり、【ケ】は電子を相手に与えて酸化される物質である。
✏️ 大問1（全4問）
問1文中の【ア】（酸化と化合するもの）として正しいものはどれか。
正解はイ：酸素。酸化の定義①：酸素と化合する。例：2Mg + O₂ → 2MgO。
問2文中の【ウ】（酸化で増え、還元で減るもの）として正しいものはどれか。
正解はア：酸化数。酸化 = 酸化数が増える、還元 = 酸化数が減る。これが最も基本的かつ広い定義です。
問3文中の【ク】（電子を受け取り還元される物質）として正しいものはどれか。
正解はウ：酸化剤。酸化剤は「電子を受け取る（自身は還元される）」物質です。覚え方：「酸化剤」は相手を「酸化」させるが、自身は「還元」される。
問4CuSO₄ 水溶液に亜鉛板を浸すと Cu²⁺ が析出する。Cu²⁺ の酸化数の変化として正しいものはどれか。
正解はイ：+2 → 0。Cu²⁺（+2）は電子を 2個受け取り Cu（単体、0）になります。酸化数が減少 = 還元されました。Zn は 0 → +2 と酸化されます。
2大問2：酸化還元反応の識別と酸化剤・還元剤 〈岩手大・防衛大改〉
🏆 岩手大・防衛大 入試問題（改題）
次の反応(ア)〜(カ)のうち、酸化還元反応を選べ。また酸化剤・還元剤を答えよ。
(ア) SiO₂ + 2NaOH → Na₂SiO₃ + H₂O
(イ) H₂O₂ + 2KI + 2HCl → 2H₂O + I₂ + 2KCl
(ウ) NH₃ + NaCl + CO₂ + H₂O → NaHCO₃ + NH₄Cl
(エ) 2AgNO₃ + 2NaOH → Ag₂O + H₂O + 2NaNO₃
(オ) 2Al + 6HCl → 2AlCl₃ + 3H₂
(カ) Pb + PbO₂ + 2H₂SO₄ → 2PbSO₄ + 2H₂O
✏️ 大問2（全4問）
問1酸化還元反応でない反応はどれとどれか。
正解はエ：(ア)(ウ)(エ)。
(ア)：すべての元素の酸化数が変化なし。
(ウ)：N:−3, H:+1, Na:+1, Cl:−1, C:+4, O:−2 とも変化なし。
(エ)：Ag:+1, N:+5, O:−2, Na:+1 とも変化なし。
(イ)(オ)(カ)は酸化還元反応です。
問2反応(イ)の酸化剤と還元剤の正しい組み合わせはどれか。
正解はイ。H₂O₂ の O：−1 → −2（減 = 還元された = 酸化剤）、KI の I：−1 → 0（増 = 酸化された = 還元剤）。
問3反応(オ) 2Al + 6HCl → 2AlCl₃ + 3H₂ で、Al の酸化数の変化として正しいものはどれか。
正解はウ：0 → +3。Al は単体（酸化数 0）→ AlCl₃ 中（Cl が −1 なので Al は +3）。酸化されているので Al が還元剤。
問4反応(カ) Pb + PbO₂ + 2H₂SO₄ → 2PbSO₄ + 2H₂O で、酸化剤はどれか。
正解はア：PbO₂。PbO₂ の Pb：+4 → PbSO₄ の Pb：+2（減 = 還元された = 酸化剤）。Pb（単体）の Pb：0 → +2（増 = 酸化された = 還元剤）。これは鉛蓄電池の放電反応です。
3大問3：KMnO₄ 滴定計算 〈大学入試・標準〉
🏆 大学入試 標準問題
0.252g のシュウ酸結晶 (COOH)₂・2H₂O（分子量 126）を水に溶かして 100 mL の溶液にし、その中から 25.0 mL を取り、希硫酸で酸性にした。これを温めながら濃度不明の過マンガン酸カリウム水溶液を少量ずつ加えていくと、20.0 mL 加えたところで反応が完了した。
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O …①
(COOH)₂ → 2CO₂ + 2H⁺ + 2e⁻ …②
✏️ 大問3（全4問）
問1100 mL 溶液を作るときに使う器具として正しいものはどれか。
正解はイ：メスフラスコ。正確に決まった体積（ここでは 100 mL）の溶液を調製するときはメスフラスコを使います。25.0 mL を正確に取る器具はホールピペットです。
問2100 mL 中に含まれるシュウ酸 (COOH)₂ の物質量はいくらか。（(COOH)₂・2H₂O の分子量 = 126）
正解はウ：2.00×10⁻³ mol。
シュウ酸の物質量 = 0.252 g ÷ 126 g/mol = 2.00×10⁻³ mol
問325.0 mL 中のシュウ酸から得られる電子の物質量はいくらか。
正解はイ：1.00×10⁻³ mol。
25.0 mL 中の (COOH)₂ = 2.00×10⁻³ mol × (25.0/100) = 5.00×10⁻⁴ mol
電子の物質量 = 5.00×10⁻⁴ × 2 = 1.00×10⁻³ mol
問4この過マンガン酸カリウム水溶液のモル濃度はいくらか。
正解はア：1.00×10⁻² mol/L。
受け取る e⁻ = 与えた e⁻ より
KMnO₄ の濃度を x mol/L とすると：
x × (20.0/1000) × 5 = 1.00×10⁻³
x × 0.020 × 5 = 1.00×10⁻³
x = 1.00×10⁻³ ÷ 0.100 = 1.00×10⁻² mol/L
4大問4：過酸化水素水の濃度決定 〈甲南大・改〉
🏆 甲南大学 入試問題（改題）
濃度未知の過酸化水素水 10.0 mL をコニカルビーカーに入れ、硫酸で酸性にした。
2.00×10⁻² mol/L の KMnO₄ 水溶液をビュレットで滴下したところ、終点まで 12.0 mL を要した。
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
H₂O₂ → O₂ + 2H⁺ + 2e⁻
✏️ 大問4（全3問）
問1この反応で H₂O₂ は何としてはたらいているか。
正解はウ：還元剤。H₂O₂ の O：−1 → O₂ の O：0（増 = 酸化された = 還元剤）。KMnO₄ が酸化剤として H₂O₂ を酸化します。H₂O₂ は通常は酸化剤ですが、より強力な酸化剤（KMnO₄）と反応するときは還元剤になります。
問2KMnO₄ が受け取った e⁻ の物質量はいくらか。
正解はイ：1.20×10⁻³ mol。
KMnO₄ の物質量 = 2.00×10⁻² × 12.0/1000 = 2.40×10⁻⁴ mol
受け取る e⁻ = 2.40×10⁻⁴ × 5 = 1.20×10⁻³ mol
問3過酸化水素水のモル濃度はいくらか。（H₂O₂ 1 mol が e⁻ 2 mol を与える）
正解はエ：6.00×10⁻² mol/L。
受け取る e⁻ = 与えた e⁻
H₂O₂ の濃度を x mol/L とすると：
x × (10.0/1000) × 2 = 1.20×10⁻³
x × 0.010 × 2 = 1.20×10⁻³
x = 1.20×10⁻³ ÷ 0.020 = 6.00×10⁻² mol/L
