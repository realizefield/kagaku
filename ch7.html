🔥 1. 酸性酸化物・塩基性酸化物
酸化物には、水に溶けると酸になるものと、塩基になるものがあります。
🔴 酸性酸化物
水に溶けると酸になる酸化物。
主に非金属元素の酸化物。
CO₂ ＋ H₂O → H₂CO₃
SO₂ ＋ H₂O → H₂SO₃
SO₃ ＋ H₂O → H₂SO₄
NO₂ ＋ H₂O → HNO₃ など
代表例：CO₂・SO₂・SO₃・NO₂・SiO₂
💡 塩基とも反応して塩と水を生じます。
CO₂ ＋ 2NaOH → Na₂CO₃ ＋ H₂O
🟢 塩基性酸化物
水に溶けると塩基になる酸化物。
主に金属元素の酸化物。
Na₂O ＋ H₂O → 2NaOH
CaO ＋ H₂O → Ca(OH)₂
代表例：Na₂O・CaO・CuO・Fe₂O₃
💡 酸とも反応して塩と水を生じます。
CaO ＋ 2HCl → CaCl₂ ＋ H₂O
📌 両性酸化物
Al・Zn・Sn・Pb の酸化物は酸にも強塩基にも反応します。→ 両性酸化物
Al₂O₃ ＋ 6HCl → 2AlCl₃ ＋ 3H₂O　（酸と反応）
Al₂O₃ ＋ 2NaOH → 2NaAlO₂ ＋ H₂O　（強塩基と反応）

① 金属の酸化物が「塩基性」になる理由
金属（例：Na、Ca）は電気陰性度がとても小さいです。
そのため、O との結合では電子がほぼ O 側に引き取られてしまいます。

Na δ⁺ ─── O δ⁻ 　→　実質ほぼ Na⁺ と O²⁻ がくっついた状態
このような酸化物が水と出会うと…
O²⁻ が水の H⁺ をもらって OH⁻（水酸化物イオン） を生じます。
Na₂O ＋ H₂O → 2Na⁺ ＋ 2OH⁻ → 2NaOH
→ OH⁻ が出るから塩基性になります！
② 非金属の酸化物が「酸性」になる理由
非金属（例：C、S）は電気陰性度が比較的大きいです。
O との電気陰性度の差が金属ほど大きくないので、結合は共有結合的になります。

O ＝ C ＝ O　（CO₂）　← C と O が電子を共有している
このような酸化物が水と反応すると…
C=O 結合の C が δ⁺ になっているので、水の O が C に近づき、
最終的に H⁺（水素イオン） が放出されます。
CO₂ ＋ H₂O → H₂CO₃ → H⁺ ＋ HCO₃⁻
→ H⁺ が出るから酸性になります！
③ 両性酸化物（Al・Zn・Sn・Pb）が「どちらにもなれる」理由
Al・Zn・Sn・Pb は電気陰性度が金属と非金属の中間的な値をもっています。
そのため、O との結合はイオン結合でも共有結合でもある「中間的な性質」になります。

強酸（HCl など）と反応
金属のように振る舞い
Al³⁺ などが溶け出す
強塩基（NaOH など）と反応
非金属のように振る舞い
AlO₂⁻ などが溶け出す
✅ 「ちょうど金属と非金属の境目あたり」にいる元素なので、どちらの性質も使い分けられるのです。
🔑 まとめ
・金属の酸化物：電気陰性度が小さい → O がほぼ O²⁻ → 水と反応して OH⁻ → 塩基性
・非金属の酸化物：共有結合的 → C や S が δ⁺ → 水と反応して H⁺ → 酸性
・両性（Al・Zn など）：中間的な性質 → 相手によって酸にも塩基にもなれる
※ 入試では「金属→塩基性、非金属→酸性、Al・Zn・Sn・Pb→両性」を覚えれば十分です。このコラムはその「なぜ？」を理解するためのものです。

💧 2. 水のイオン積 Kw
純粋な水（純水）はごくわずかに電離しています。
H₂O ⇌ H⁺ ＋ OH⁻
📌 水のイオン積 Kw（25℃）
[H⁺] × [OH⁻] ＝ 1.0 × 10⁻¹⁴　(mol/L)²
※ これを水のイオン積 Kw といいます。
※ 温度が変わらない限り、酸性・中性・塩基性いずれの水溶液でも常に成り立ちます。
中性のときは H⁺ と OH⁻ の濃度が等しくなります：
[H⁺] ＝ [OH⁻] ＝ 1.0 × 10⁻⁷ mol/L　（25℃ 中性）
液性[H⁺] の大きさ[OH⁻] の大きさ
酸性[H⁺] > 1.0×10⁻⁷ mol/L[OH⁻] < 1.0×10⁻⁷ mol/L
中性[H⁺] ＝ 1.0×10⁻⁷ mol/L[OH⁻] ＝ 1.0×10⁻⁷ mol/L
塩基性[H⁺] < 1.0×10⁻⁷ mol/L[OH⁻] > 1.0×10⁻⁷ mol/L
⚡ Kw は温度により変化します。問題では特に断りがなければ 25℃（Kw = 10⁻¹⁴）として計算しましょう。
🔢 3. pH（水素イオン指数）
[H⁺] の値は 10 のマイナス何乗という形で表すと不便なので、その指数部分の数を使って表したものが pH です。
📌 pH の定義（最重要）
[H⁺] ＝ 10−n mol/L　のとき　pH ＝ n
（より正確には pH ＝ −log₁₀[H⁺] ですが、まずは「指数の絶対値 ＝ pH」で覚えましょう）
← 酸性が強い　　　　　　　　　　塩基性が強い →
pH 01234 567 89101112 1314
← 酸性（pH < 7） 中性（pH = 7） 塩基性（pH > 7）→

「pH（ペーハー）さん（酸）は 7 より小さい」
📊 pH 計算の3パターン
パターン① [H⁺] がわかっているとき
1[H⁺] ＝ 10−n の形に直す
2pH ＝ n を読み取る
【例】[H⁺] ＝ 1.0×10−4 mol/L → pH ＝ 4
【例】0.01mol/L HCl（強酸・1価）
　　 [H⁺] ＝ 0.01 ＝ 1.0×10−2 mol/L → pH ＝ 2
【例】0.005mol/L H₂SO₄（強酸・2価）
　　 [H⁺] ＝ 0.005 × 2 ＝ 0.01 ＝ 1.0×10−2 mol/L → pH ＝ 2
💡 多価の酸は「モル濃度 × 価数」が [H⁺] になります。

▶ 酸性酸化物・塩基性酸化物

水に溶けると酸になる酸化物を 酸性酸化物 といい、主に 非金属 元素の酸化物です。 代表例は CO₂・SO₂・NO₂ などです。 水に溶けると塩基になる酸化物を 塩基性酸化物 といい、主に 金属 元素の酸化物です。 代表例は Na₂O・CaO などです。 Al・Zn の酸化物のように酸にも強塩基にも反応するものを 両性酸化物 といいます。

▶ 水のイオン積と pH

25℃において [H⁺]×[OH⁻] ＝ 1.0×10⁻¹⁴ が常に成り立ちます。 これを水の イオン積 といいます。 中性のとき [H⁺] ＝ 1.0×10⁻⁷ mol/L です。 [H⁺] ＝ 10−n のとき pH ＝ n と定義します。 酸性は pH が 7より小さい、 塩基性は pH が 7より大きい です。

▶ pH の計算

0.01mol/L の塩酸（強酸・1価）の [H⁺] ＝ 1.0×10⁻² mol/L なので pH ＝ 2 です。 0.01mol/L の NaOH（強塩基・1価）の [OH⁻] ＝ 1.0×10⁻² mol/L なので [H⁺] ＝ 1.0×10⁻¹² mol/L となり pH ＝ 12 です。 弱酸で電離度が与えられている場合は [H⁺] ＝ モル濃度 × 電離度α として計算します。
