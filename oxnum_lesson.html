酸化数の定義と計算

「酸化数ってなんだろう？」というところから始めます。ルールを覚えるのではなく、まず具体例を見て感覚をつかむことが大切です。

🔰 初学者向け　丁寧解説
0そもそも「酸化数」って何？

化学の反応を学んでいると「酸化」「還元」という言葉が出てきます。
「鉄が酸化された」「銅が還元された」—— でも、これを数値で正確に表す方法はないでしょうか？

そこで生まれたのが 酸化数（さんかすう） というルールです。

酸化数とは、「その原子がどれだけ電子を失っているか（または得ているか）を示す数」です。
電子を失うほど酸化数は大きく（プラスに）、電子を得るほど小さく（マイナスに）なります。

💡 まず直感でイメージしてみよう
鉄（Fe）の単体
→
電子を誰にも渡していない → 酸化数 0基準
酸化鉄（Fe₂O₃）の中の Fe
→
O に電子を 3個奪われた → 酸化数 +3酸化された
塩化鉄（FeCl₂）の中の Fe
→
Cl に電子を 2個奪われた → 酸化数 +2酸化された
📌 酸化数が大きい（プラスが大きい）ほど、「より多く酸化されている」状態です。
　 酸化数が小さい（マイナスが大きい）ほど、「より多く電子を持っている」状態です。
★ 酸化数のポイント
  • 酸化数は 整数（…−3, −2, −1, 0, +1, +2, +3…）で表す
  • プラス の符号は省略せず必ず書く（例：+2, +3）
  • 単体（1種類の元素だけ）の酸化数は必ず 0
  • 酸化数は「原子1個あたり」の数値
1酸化数を決める5つのルール

酸化数は「実際に電子が何個移動したか」を正確に計算するのは大変なので、便利なルールを使って決めます。このルールさえ覚えれば、ほとんどの物質の酸化数が求められます。

📋 酸化数を決める5つのルール
①
単体中の原子の酸化数は 0
例：Fe = 0 ／ Cu = 0 ／ H₂ の各 H = 0 ／ O₂ の各 O = 0 ／ Cl₂ の各 Cl = 0
②
単原子イオンの酸化数 = イオンの電荷
例：Na⁺ → +1 ／ Ca²⁺ → +2 ／ Cl⁻ → −1 ／ O²⁻ → −2
③
化合物中の H = +1、O = −2（例外あり）
H は +1、O は −2 を原則として使う
⚠️ 例外：金属水素化物（NaH など）の H = −1 ／ H₂O₂ の O = −1
④
化合物中の全原子の酸化数の合計 = 0
例：H₂O → (+1)×2 + (−2) = 0 ✓ ／ CuO → (+2) + (−2) = 0 ✓
⑤
多原子イオン中の全原子の酸化数の合計 = イオンの電荷
例：SO₄²⁻ → 合計 = −2 ／ NH₄⁺ → 合計 = +1 ／ OH⁻ → 合計 = −1
⚠️ 使う順番が大切！
ルール①②は「そのまま決まる」場合。ルール③〜⑤は「他の原子の酸化数を先に決めてから、残りを計算する」のが基本です。
求めたい元素の酸化数を x として、方程式を立てるのが最もミスが少ない方法です。
✏️ ルールの確認（全4問）
問1単体である Cl₂ 中の Cl の酸化数はいくらか。
正解はイ：0。
Cl₂ は塩素原子だけからなる単体です。ルール①より、単体中の原子の酸化数は必ず 0。
「Cl は普通 −1」と混同しやすいですが、Cl₂ は単体なので 0 です。
問2Ca²⁺（カルシウムイオン）の酸化数はいくらか。
正解はウ：+2。
Ca²⁺ は単原子イオンです。ルール②より、単原子イオンの酸化数 = イオンの電荷 = +2。
問3H₂O（水）中の O の酸化数は？（H = +1 として）
正解はア：−2。
H₂O の合計 = 0（ルール④）。H = +1 が 2個なので (+1)×2 + O の酸化数 = 0 → O = −2。
あるいはルール③をそのまま使っても O = −2 です。
問4OH⁻（水酸化物イオン）の酸化数の合計はいくらか。
正解はエ：−1。
OH⁻ は多原子イオン（電荷 −1）です。ルール⑤より、多原子イオン中の酸化数の合計 = イオンの電荷 = −1。
実際に確認：O = −2、H = +1 → 合計 = −2 + 1 = −1 ✓
2酸化数の計算手順（x をおいて解く）

求めたい元素の酸化数を x とおいて方程式を立てるのが、最もミスの少ない方法です。

手順は次の通りです：
① わかっている元素（H = +1、O = −2 など）を先に書く
② 求めたい元素を x とおく
③ 「合計 = 0（化合物）または合計 = 電荷（イオン）」の式を立てる
④ x を解く

📝 例題1：NH₃ 中の N の酸化数
①
NH₃ は化合物なので、酸化数の合計 = 0
②
H = +1 はルール③でわかる。N の酸化数を x とおく
③
式を立てる：
x + (+1) × 3 = 0
④
x = 0 − 3 = −3
→ N の酸化数は −3
📝 例題2：H₂SO₄ 中の S の酸化数
①
H₂SO₄ は化合物なので、酸化数の合計 = 0
②
H = +1、O = −2 はルール③でわかる。S の酸化数を x とおく
③
式を立てる（H が 2個、O が 4個）：
(+1) × 2 + x + (−2) × 4 = 0 2 + x − 8 = 0
④
x = 8 − 2 = +6
→ S の酸化数は +6
📝 例題3：MnO₄⁻（過マンガン酸イオン）中の Mn の酸化数
①
MnO₄⁻ は多原子イオン（電荷 −1）なので、酸化数の合計 = −1（ルール⑤）
②
O = −2 はルール③でわかる。Mn の酸化数を x とおく
③
式を立てる（O が 4個）：
x + (−2) × 4 = −1 x − 8 = −1
④
x = −1 + 8 = +7
→ Mn の酸化数は +7
📝 例題4：Cr₂O₇²⁻（ニクロム酸イオン）中の Cr の酸化数
①
Cr₂O₇²⁻ は多原子イオン（電荷 −2）なので、酸化数の合計 = −2
②
O = −2。Cr の酸化数を x とおく。Cr は 2個あるので 2x になることに注意！
③
式を立てる（Cr が 2個、O が 7個）：
2x + (−2) × 7 = −2 2x − 14 = −2
④
2x = 12 → x = +6
→ Cr の酸化数は +6
⚠️ Cr₂O₇²⁻ のような「同じ元素が複数個ある」式は要注意
Cr が 2個あるので、方程式は「2x + ⋯ = −2」と 係数をかけてから x を求めます。
「x = −2 − (−14) = 12」と先に出てしまう間違いが多いので注意しましょう。
✏️ 計算練習（全5問）
問1HCl 中の Cl の酸化数はいくらか。（H = +1）
正解はイ：−1。
HCl は化合物、合計 = 0。x を Cl の酸化数とすると (+1) + x = 0 → x = −1。
単体 Cl₂（酸化数 0）と混同しないように！
問2NO₂ 中の N の酸化数はいくらか。（O = −2）
正解はウ：+4。
x + (−2)×2 = 0 → x − 4 = 0 → x = +4。
N の酸化数は −3 〜 +5 と幅広いので、毎回 x を使って計算しましょう。
問3SO₃²⁻（亜硫酸イオン）中の S の酸化数はいくらか。（O = −2）
正解はエ：+4。
SO₃²⁻ は多原子イオン（電荷 −2）。x + (−2)×3 = −2 → x − 6 = −2 → x = +4。
合計が「0」でなく「−2」になることに注意！
問4KMnO₄ 中の Mn の酸化数はいくらか。（K = +1、O = −2）
正解はア：+7。
(+1) + x + (−2)×4 = 0 → 1 + x − 8 = 0 → x = +7。
KMnO₄ は酸化剤として重要。Mn の酸化数 +7 は必ず覚えましょう。
問5K₂Cr₂O₇ 中の Cr の酸化数はいくらか。（K = +1、O = −2）
正解はイ：+6。
(+1)×2 + 2x + (−2)×7 = 0 → 2 + 2x − 14 = 0 → 2x = 12 → x = +6。
Cr が 2個あるので「2x」になる点がポイント！
3例外をおさえよう（H₂O₂・NaH・OF₂）

ルール③では「化合物中の H = +1、O = −2」と学びましたが、例外がいくつかあります。入試でも狙われるポイントなので、しっかり確認しましょう。

⚠️ 例外一覧
H₂O₂
O の酸化数は
−1
（通常の −2 ではない！）
確認：(+1)×2 + x×2 = 0 → x = −1。O−O 結合があるため酸化数が上がります。
NaH・CaH₂
H の酸化数は
−1
（通常の +1 ではない！）
金属水素化物では、Na より H の方が電気陰性度が大きいため、H が電子を引き寄せて −1 になります。
確認（NaH）：(+1) + x = 0 → x = −1。
OF₂
O の酸化数は
+2
（F が −1 で O を押しのける）
F（フッ素）は全元素中で最も電気陰性度が大きいため、OF₂ では O が電子を奪われてプラスになります。
確認：x + (−1)×2 = 0 → x = +2。（F は常に −1）
💡 なぜ例外が生まれるのか？
📌 酸化数のルール③は「H = +1、O = −2 がほとんどの場合に成り立つ」という経験則です。

例外が起きるのは、結合相手の電気陰性度の大小関係が変わるときです。
・NaH：Na よりも H の方が電気陰性度が大きい → H が電子を得る（−1）
・OF₂：F の電気陰性度が O より大きい → O が電子を失う（+2）
・H₂O₂：O−O 結合のため電子を分け合う → O が −1 どまり

ルール③の例外に出会ったら、x をおいて計算する方法で確認するのが確実です。
✏️ 例外の確認（全3問）
問1H₂O₂ 中の O の酸化数はいくらか。（H = +1）
正解はウ：−1。
x を O の酸化数とすると (+1)×2 + x×2 = 0 → 2 + 2x = 0 → x = −1。
H₂O₂ の O は例外で −1 です（H₂O の O = −2 と混同しないこと！）。
問2NaH（水素化ナトリウム）中の H の酸化数はいくらか。（Na = +1）
正解はエ：−1。
(+1) + x = 0 → x = −1。
NaH は金属水素化物。H は通常 +1 ですが、金属（Na, Ca など）と結合するときは例外的に −1 になります。
問3OF₂ 中の O の酸化数はいくらか。（F = −1）
正解はイ：+2。
x + (−1)×2 = 0 → x = +2。
F は電気陰性度が最大で常に −1。O と結合すると O の方が電子を奪われて珍しくプラスになります。
4よくある間違いとその対策
🚫 よくある間違いパターン
✗ 間違い①：「Cl₂ の Cl は −1 だ」
✓ 正解：Cl₂ は単体なのでルール①より 0。HCl の Cl が −1 なのと混同している。
✗ 間違い②：「SO₄²⁻ の合計は 0 だ」
✓ 正解：SO₄²⁻ はイオン（電荷 −2）なので合計 = −2（ルール⑤）。合計が 0 になるのは化合物のみ。
✗ 間違い③：「Cr₂O₇²⁻ で 2x + (−2)×7 = −2 → 2x = 12 → x = 12」
✓ 正解：2x = 12 までは正しい。次に x = 6 と計算する（2 で割るのを忘れない！）。
✗ 間違い④：「H₂O₂ の O は −2 だ」
✓ 正解：H₂O₂ の O は例外で −1。x をおいて計算すれば確実に確認できる。
✗ 間違い⑤：「酸化数にプラス符号を書かなかった」
✓ 正解：酸化数がプラスのときは +2、+6 のように + 符号を必ず書く。「2」だけでは不正確。
✏️ 間違いを見つけよう（全3問）
問1「H₂ の H の酸化数は +1 だ」この説明の誤りはどこか。
正解はウ。H₂ は水素の単体なのでルール①より酸化数 = 0。
「H の酸化数は +1」は化合物（H₂O, HCl など）の場合のルールで、H₂ には適用しません。
問2「NO₃⁻ の合計 = 0 として N の酸化数を計算したら +6 になった」どこが間違いか。
正解はア。NO₃⁻ は多原子イオン（電荷 −1）なので合計 = −1。
正しくは x + (−2)×3 = −1 → x = +5。合計を 0 にしてしまうのが最頻出ミスです。
問3次のうち、酸化数の表記として正しいものはどれか。
正解はエ：+6。
酸化数がプラスのときは + 符号を必ず書きます。
ア〜ウは符号が抜けています（正しくはそれぞれ +2、+7、−3）。
