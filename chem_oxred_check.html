1酸化数の計算
📖 江原予備校テキスト 酸化還元(1) より
酸化数のルールを使って、下線部の元素の酸化数を求める問題です。
✏️ 練習1：酸化数を求めよ
問1CuSO₄ 中の S の酸化数はいくらか。Cu = +2, O = −2 とする。
正解はイ：+6。Cu の酸化数は +2、O×4 は −8。全体 = 0 なので (+2) + x + (−2)×4 = 0 → x = +6。
問2KClO₃ 中の Cl の酸化数はいくらか。K = +1, O = −2 とする。
正解はエ：+5。(+1) + x + (−2)×3 = 0 → x = +5。
問3NO₂ 中の N の酸化数はいくらか。O = −2 とする。
正解はア：+4。x + (−2)×2 = 0 → x = +4。
問4Cr₂O₇²⁻（ニクロム酸イオン）中の Cr の酸化数はいくらか。O = −2 とする。
正解はウ：+6。多原子イオンなので 2x + (−2)×7 = −2 → 2x = 12 → x = +6。K₂Cr₂O₇ も同様に +6。
2酸化還元反応の識別
📖 江原予備校テキスト 酸化還元(1) 練習より
与えられた反応が酸化還元反応かどうか、また酸化剤・還元剤を識別する問題です。
✏️ 練習2：酸化還元反応の判断
問1次の反応のうち、酸化還元反応でないものはどれか。
正解はウ。H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O は中和反応で、全元素の酸化数が変化していません。アはAl:0→+3、イはCl:0→−1, O:−1→0、エはN:+4→+5,+2と変化しており、いずれも酸化還元反応。
問2反応 H₂O₂ + 2KI + 2HCl → 2H₂O + I₂ + 2KCl で酸化剤はどれか。
正解はイ：H₂O₂。O：−1 → −2（減 = 還元された = 酸化剤）。KI の I：−1 → 0（増 = 酸化された = 還元剤）。
問3反応 3NO₂ + H₂O → 2HNO₃ + NO で、酸化された N の酸化数の変化として正しいものはどれか。
正解はエ：+4 → +5。NO₂ の N は +4。HNO₃ の N は +5（酸化）、NO の N は +2（還元）。この反応では同じ NO₂ が酸化剤にも還元剤にもなっています（自己酸化還元）。
問4反応 2H₂O₂ → 2H₂O + O₂ で、H₂O₂ は何としてはたらいているか。
正解はイ。H₂O₂ の O：−1 → H₂O の O：−2（還元 → 酸化剤）と O₂ の O：0（酸化 → 還元剤）。同じ H₂O₂ 分子が分解して酸化・還元両方が起こります（不均化反応）。
3半反応式とイオン反応式
📖 江原予備校テキスト 酸化還元(1) より
重要な酸化剤の半反応式を完成させる問題です。
✏️ 練習3：半反応式の完成
問1希硝酸（HNO₃）が酸化剤としてはたらくときの半反応式として正しいものはどれか。
正解はウ。希硝酸は NO（一酸化窒素）を生成します（濃硝酸は NO₂）。N：+5 → +2（変化 3）→ e⁻ 3個。O を補うために右辺に 2H₂O、左辺に 4H⁺。
問2シュウ酸（(COOH)₂）が還元剤としてはたらくときの半反応式として正しいものはどれか。
正解はア。シュウ酸の C は +3、CO₂ の C は +4（増加 → 酸化 → 還元剤）。変化×2 = 2e⁻ を右辺に。H は H⁺ として右辺に 2個。1mol の (COOH)₂ から 2molの電子を与えます。
問3KMnO₄ と H₂O₂ が反応するとき（KMnO₄ が酸化剤）、e⁻ を消すために何倍するか。
正解はエ（ウとエは同じ内容です）。KMnO₄ 側は e⁻ が 5個（Mn：+7→+2）、H₂O₂ 側は e⁻ が 2個（O：−1→0）。最小公倍数は 10。KMnO₄ 側 ×2 で e⁻ 10個、H₂O₂ 側 ×5 で e⁻ 10個。
問4イオン反応式を作るとき、酸化剤と還元剤の半反応式を足した後は、両辺に共通して存在するイオンや分子を消去することができる。
○ 正解。e⁻ を消すだけでなく、両辺に共通する H⁺ や H₂O なども整理（消去または計算）してすっきりしたイオン反応式を作ります。
4酸化還元滴定の基礎
📖 江原予備校テキスト 酸化還元(1) 酸化還元滴定より
「与えた電子の量 = 受け取った電子の量」の関係を使って濃度を計算する問題です。
✏️ 練習4：酸化還元滴定の計算
問1酸化還元滴定の計算で使う基本原理はどれか。
正解はイ。酸化還元滴定では「酸化剤が受け取る e⁻ の mol = 還元剤が与える e⁻ の mol」を使います。これが計算の基本式です。
問20.020 mol/L の KMnO₄ 水溶液 10.0 mL と過不足なく反応するシュウ酸 (COOH)₂ の物質量はいくらか。（KMnO₄ は e⁻ 5個受け取り、(COOH)₂ は e⁻ 2個与える）
正解はウ：5.0×10⁻⁴ mol。
KMnO₄ が受け取る e⁻ = 0.020 mol/L × 0.010 L × 5 = 1.0×10⁻³ mol
(COOH)₂ の物質量 = χ、与える e⁻ = χ × 2
1.0×10⁻³ = χ × 2 → χ = 5.0×10⁻⁴ mol
問3過マンガン酸カリウム滴定で「終点」を判断する方法として正しいものはどれか。
正解はア。KMnO₄ は赤紫色。還元剤が存在する間は反応して脱色されますが、還元剤がなくなると次の1滴の赤紫色が消えずに残ります。これを終点とします。
問4過マンガン酸カリウム（KMnO₄）滴定の際、酸性にするために塩酸（HCl）を使うことができる。
× 不正解。塩酸の Cl⁻ が KMnO₄ に対して還元剤としてはたらくため、誤差が生じます。硫酸（H₂SO₄）を使って酸性にします（硝酸も HNO₃ が酸化剤のため不可）。
