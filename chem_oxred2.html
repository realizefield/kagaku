化学 / 酸化還元(2)

酸化還元(2) ★★★

ヨウ素滴定・チオ硫酸ナトリウム滴定・COD・DO を学びます。計算の「mol の橋渡し」を理解するのがこの単元の核心です。

1ヨウ素滴定の原理
📖 説明

ヨウ素滴定とは、ヨウ素（I₂）を酸化剤または還元剤として利用し、その量をチオ硫酸ナトリウム（Na₂S₂O₃）で逆滴定して目的物質の濃度を求める方法です。

▶ 基本の3試薬とその役割

試薬役割電子の授受半反応式
I₂（ヨウ素） 酸化剤 e⁻ を 2mol 受け取る I₂ + 2e⁻ → 2I⁻
SO₂（二酸化硫黄） 還元剤 e⁻ を 2mol 渡す SO₂ + 2H₂O → SO₄²⁻ + 4H⁺ + 2e⁻
Na₂S₂O₃（チオ硫酸Na） 還元剤（逆滴定用） e⁻ を 1mol 渡す 2S₂O₃²⁻ → S₄O₆²⁻ + 2e⁻
⚠️ チオ硫酸Naは1molで電子1mol！
  • 2S₂O₃²⁻ → S₄O₆²⁻ + 2e⁻ だから、S₂O₃²⁻ 1mol あたり e⁻ 1mol を渡す
  • I₂ は 1mol で e⁻ を 2mol 受け取る
  • よって I₂ 1mol ⇔ Na₂S₂O₃ 2mol（2:1 の関係）

▶ 指示薬：デンプン

ヨウ素（I₂）はデンプンと反応して青紫色になります。チオ硫酸ナトリウムを加えていくと I₂ が消費されて脱色されます。

状態色
I₂ が残っている（滴定途中）青紫色
I₂ が全部消費された（終点）無色

→ 青紫色から無色への変化が終点のサイン

★ 最重要の mol 関係
  • I₂ 1mol ＝ e⁻ 2mol（I₂ は酸化剤）
  • Na₂S₂O₃ 1mol ＝ e⁻ 1mol（チオ硫酸Naは還元剤）
  • ∴ I₂ 1mol ⇔ Na₂S₂O₃ 2mol
  • SO₂ 1mol ＝ e⁻ 2mol → SO₂ 1mol ⇔ I₂ 1mol
✏️ ミニテスト（全3問）
問1チオ硫酸ナトリウム（Na₂S₂O₃）1mol が反応で授受する電子は何 mol か。
正解はイ：1mol。半反応式 2S₂O₃²⁻ → S₄O₆²⁻ + 2e⁻ より、S₂O₃²⁻（＝Na₂S₂O₃）1mol につき e⁻ 1mol を渡します。I₂ は 1mol で e⁻ 2mol を受け取るので、I₂ 1mol ⇔ Na₂S₂O₃ 2mol。
問2ヨウ素滴定でデンプンを指示薬に使うとき、終点は「青紫色から無色」への変化である。
○ 正解。I₂ が存在するとデンプンと反応して青紫色になります。チオ硫酸 Na で I₂ が消費されると脱色されて無色になります。この変化が終点です。
問3I₂ 1mol と過不足なく反応する Na₂S₂O₃ は何 mol か。
正解はウ：2mol。I₂ 1mol は e⁻ 2mol を受け取る。Na₂S₂O₃ は 1mol で e⁻ 1mol を渡す。受け取る e⁻ = 渡す e⁻ より 1×2 = 2×1 → Na₂S₂O₃ は 2mol 必要。
2ヨウ素滴定の計算手順（バックタイトレーション）
📖 説明

ヨウ素滴定では「直接滴定できない量を、過剰の I₂ と反応させてから残った I₂ を Na₂S₂O₃ で逆滴定」します。これをバックタイトレーション（逆滴定）といいます。

▶ 【例題1】の解き方：SO₂ の定量

0.20 mol/L ヨウ素溶液 25 mL に SO₂ を完全反応させ、残った I₂ を 0.050 mol/L Na₂S₂O₃ 溶液で滴定したところ 20 mL を要した。SO₂ の物質量を求めよ。

I₂（全量）
0.20×0.025
= 5.0×10⁻³ mol
→
SO₂ と反応
SO₂ に使われた I₂
= SO₂ の mol と同じ
（1:1 の関係）
＋
残った I₂
Na₂S₂O₃ で逆滴定
0.050×0.020÷2
= 5.0×10⁻⁴ mol
①
I₂ の全物質量を計算する
0.20 mol/L × 25/1000 L = 5.0×10⁻³ mol
②
残った I₂（Na₂S₂O₃ と反応した I₂）を計算する
I₂ 1mol ⇔ Na₂S₂O₃ 2mol より
0.050 × 20/1000 × 1/2 = 5.0×10⁻⁴ mol
③
SO₂ と反応した I₂ = 全量 − 残量
5.0×10⁻³ − 5.0×10⁻⁴ = 4.5×10⁻³ mol
④
SO₂ 1mol ⇔ I₂ 1mol（どちらも e⁻ 2mol の授受）
∴ SO₂ の物質量 = 4.5×10⁻³ mol
★ バックタイトレーションの鉄則
  • SO₂ に使われた I₂ の mol ＝ 全 I₂ の mol − 残った I₂ の mol
  • 残った I₂ の mol ＝ Na₂S₂O₃ の mol ÷ 2（I₂ : Na₂S₂O₃ = 1 : 2）
  • SO₂ と I₂ は 1 : 1 で反応（どちらも e⁻ 2mol の授受）
✏️ ミニテスト（全3問）
問1ヨウ素滴定で「残った I₂ の mol」を求めるとき、Na₂S₂O₃ の mol をいくつで割るか。
正解はア：2。I₂ 1mol に Na₂S₂O₃ 2mol が対応するので、Na₂S₂O₃ の mol を 2 で割れば I₂ の mol が求まります。
問2SO₂ 1mol と反応する I₂ は何 mol か。
正解はイ：1mol。SO₂ は e⁻ 2mol を渡す還元剤、I₂ は e⁻ 2mol を受け取る酸化剤。よって SO₂ : I₂ = 1 : 1 で反応します。
問3例題1（上記）で、全 I₂ の mol はいくらか。
正解はエ：5.0×10⁻³ mol。0.20 mol/L × 25/1000 L = 5.0×10⁻³ mol。
3H₂O₂ のヨウ素滴定（間接法）
📖 説明

過酸化水素（H₂O₂）はそのままでは I₂ で直接滴定できません。そこで KI を加えて I₂ を遊離させ、その I₂ を Na₂S₂O₃ で滴定する「間接法」を使います。

▶ 反応の流れ

H₂O₂
酸化剤としてはたらく
e⁻ 2mol 受け取る
→
KI を加える
I₂ が遊離
H₂O₂ 1mol
⇔ I₂ 1mol
→
Na₂S₂O₃ で滴定
I₂ を定量
I₂ 1mol
⇔ Na₂S₂O₃ 2mol

▶ 半反応式

（酸化剤）H₂O₂ + 2H⁺ + 2e⁻ → 2H₂O
（還元剤）2I⁻ → I₂ + 2e⁻
→ H₂O₂ 1mol が I₂ 1mol を生じる（1 : 1 の関係）

▶ mol の橋渡し（計算の核心）

物質→物質mol の関係
H₂O₂⇔I₂1 : 1
I₂⇔Na₂S₂O₃1 : 2
H₂O₂⇔Na₂S₂O₃1 : 2

→ H₂O₂ 1mol ⇔ Na₂S₂O₃ 2mol が計算のキー！

▶ 計算式

H₂O₂ の濃度を x mol/L、サンプル体積を V L とすると：

x × V × 2 = Na₂S₂O₃ の mol × 1
（左辺：H₂O₂ が渡せる電子の mol ／ 右辺：Na₂S₂O₃ が渡した電子の mol）
★ 覚えよう！
  • H₂O₂ に KI を加える → I₂ が遊離（H₂O₂ が酸化剤、I⁻ が還元剤）
  • H₂O₂ : I₂ = 1 : 1
  • I₂ : Na₂S₂O₃ = 1 : 2
  • ∴ H₂O₂ : Na₂S₂O₃ = 1 : 2
✏️ ミニテスト（全3問）
問1H₂O₂ のヨウ素滴定（間接法）で、KI を加えると何が遊離するか。
正解はイ：I₂。H₂O₂ が酸化剤として KI の I⁻ を酸化し、I₂ が遊離します。この I₂ をデンプンを指示薬として Na₂S₂O₃ で滴定します。
問2H₂O₂ 1mol に対して反応する Na₂S₂O₃ は何 mol か。
正解はウ：2mol。H₂O₂ 1mol → I₂ 1mol → Na₂S₂O₃ 2mol。2段階の mol 変換をつなぐと H₂O₂ : Na₂S₂O₃ = 1 : 2。
問3H₂O₂ のヨウ素滴定（間接法）では、H₂O₂ は還元剤としてはたらく。
× 誤り。この反応では H₂O₂ は酸化剤としてはたらき、KI の I⁻ を I₂ に酸化します。（KMnO₄ との反応のように強力な酸化剤と反応するときは還元剤になります。）
4COD（化学的酸素要求量）
📖 説明

COD（Chemical Oxygen Demand）とは、試料水 1L 中の有機物を酸化分解するのに必要な酸化剤の量を、酸素（O₂）の質量（mg）に換算したものです。水質汚染の指標として使われます。

▶ 測定の手順（KMnO₄ を使う方法）

操作Ⅰ
試料水に
過剰の KMnO₄ を加え
有機物を酸化
→
操作Ⅱ
残った KMnO₄ に
シュウ酸 Na を加えて
過不足なく反応
→
計算
有機物に使われた
KMnO₄ の mol を求め
O₂ の質量に換算

▶ KMnO₄ とシュウ酸の mol 関係

MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O　（KMnO₄：e⁻ 5mol 受け取る）
C₂O₄²⁻ → 2CO₂ + 2e⁻　（シュウ酸：e⁻ 2mol 渡す）
KMnO₄ : シュウ酸 Na = 2 : 5（e⁻ を 10mol で揃える）

▶ KMnO₄ から O₂ への換算

有機物と反応した KMnO₄ の mol を O₂ の mol に換算します。

O₂ + 4H⁺ + 4e⁻ → 2H₂O　（O₂：e⁻ 4mol 受け取る）
KMnO₄ は e⁻ 5mol 受け取る、O₂ は e⁻ 4mol 受け取る
∴ KMnO₄ 1mol ⇔ O₂ 5/4 mol

▶ 計算のまとめ

計算ステップ計算式
①有機物に使われた KMnO₄ の mol全 KMnO₄ mol − シュウ酸 Na と反応した KMnO₄ mol
②対応する O₂ の molKMnO₄ の mol × 5/4
③O₂ の質量（mg）→ CODO₂ の mol × 32 × 1000 × (1000 mL / 試料 mL)
⚠️ シュウ酸 Na の扱いに注意
  • シュウ酸（(COOH)₂）とシュウ酸ナトリウム（Na₂C₂O₄）は同じ e⁻ 2mol を渡す
  • KMnO₄ : シュウ酸 = 2 : 5（mol 比）
  • シュウ酸 1mol ⇔ KMnO₄ 2/5 mol
✏️ ミニテスト（全3問）
問1COD とは何を表す指標か。
正解はア。COD は Chemical Oxygen Demand（化学的酸素要求量）。試料水 1L 中の有機物を酸化分解するのに必要な酸化剤の量を、O₂ の質量（mg）に換算したものです。値が大きいほど汚染が進んでいます。
問2KMnO₄ 1mol が受け取る e⁻ の mol と、O₂ 1mol が受け取る e⁻ の mol はそれぞれいくらか。
正解はウ。KMnO₄：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O（e⁻ 5mol）。O₂：O₂ + 4H⁺ + 4e⁻ → 2H₂O（e⁻ 4mol）。よって KMnO₄ 1mol ⇔ O₂ 5/4 mol。
問3KMnO₄ とシュウ酸 Na（Na₂C₂O₄）の mol 比（KMnO₄ : シュウ酸 Na）はいくらか。
正解はイ：2 : 5。KMnO₄ は e⁻ 5mol 受け取り、シュウ酸 Na は e⁻ 2mol 渡す。e⁻ 10mol で揃えると KMnO₄ 2mol : シュウ酸 Na 5mol。
5DO（溶存酸素）
📖 説明

DO（Dissolved Oxygen：溶存酸素）とは、水中に溶け込んでいる O₂ の量を mg/L で表したものです。

COD が「有機物の量」を表すのに対し、DO は「水中の酸素の量」を表します。DO が低い＝水が汚染されている（有機物が多く、酸素が消費されている）。

▶ DO の測定手順（ウィンクラー法）

① Mn(OH)₂ を生成
MnSO₄ + 2NaOH
→ Mn(OH)₂ + Na₂SO₄
→
② O₂ と反応
2Mn(OH)₂ + O₂
→ 2H₂MnO₃
（O₂ 1mol → H₂MnO₃ 2mol）
→
③ KI でヨウ素を遊離
H₂MnO₃ + 2KI + 2H₂SO₄
→ … + I₂
（H₂MnO₃ 1mol → I₂ 1mol）
→
④ Na₂S₂O₃ で滴定
I₂ + 2Na₂S₂O₃
→ 2NaI + Na₂S₄O₆
（I₂ 1mol ⇔ Na₂S₂O₃ 2mol）

▶ mol の連鎖（DO 計算の核心）

各ステップの mol 関係をつなぐと：

物質mol 関係
O₂ 1mol→ H₂MnO₃ 2mol（②式）
H₂MnO₃ 2mol→ I₂ 2mol（③式：1:1）
I₂ 2mol→ Na₂S₂O₃ 4mol（④式：1:2）
O₂ 1mol→ Na₂S₂O₃ 4mol（まとめ）

→ O₂ : Na₂S₂O₃ = 1 : 4 が DO 計算のキー！

▶ DO の計算式

O₂ の mol ＝ Na₂S₂O₃ の mol ÷ 4
DO（mg/L）＝ O₂ の mol × 32 × 1000 ÷ 試料量（L）
O₂ の分子量 = 32、1000 は g → mg への換算
★ COD と DO の違いを整理
  • COD：有機物を酸化するのに必要な O₂ の量（汚染の大きさ）→ 大きいほど汚染が進んでいる
  • DO：水中に実際に溶けている O₂ の量 → 小さいほど汚染が進んでいる
  • どちらも最終的に O₂ の mg/L で表す
✏️ ミニテスト（全4問）
問1DO の測定（ウィンクラー法）で O₂ 1mol に対応する Na₂S₂O₃ は何 mol か。
正解はエ：4mol。O₂ 1mol → H₂MnO₃ 2mol → I₂ 2mol → Na₂S₂O₃ 4mol と連鎖します。O₂ : Na₂S₂O₃ = 1 : 4。
問2DO が低い水は、一般的にどのような状態か。
正解はイ。有機物が多いと、それを分解する微生物が O₂ を消費するため DO が低下します。DO が低いほど水質が悪いことを示します。
問3COD が大きい水ほど、きれいな水である。
× 誤り。COD が大きいほど有機物が多く、汚染が進んでいる水です。COD は小さいほど水質がよいことを示します。
問4DO の測定で最終的に Na₂S₂O₃ で滴定する際の指示薬と終点の変化は何か。
正解はウ。Na₂S₂O₃ で I₂ を滴定するときはデンプンを指示薬として使います。I₂ が残っている間は青紫色、全て消費されると無色になります。
