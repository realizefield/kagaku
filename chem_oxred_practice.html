練習問題：酸化還元(1) ★★☆

酸化数・酸化還元の定義・酸化剤と還元剤の基礎を確認します。全10問。すぐに答え合わせできます。

✏️ 練習問題（全10問）
問1Fe₂O₃ 中の Fe の酸化数として正しいものはどれか。O = −2 とする。
正解はイ：+3。Fe の酸化数を x とすると、x×2 + (−2)×3 = 0 → 2x = 6 → x = +3。
問2MnO₄⁻（過マンガン酸イオン）中の Mn の酸化数はいくらか。
正解はウ：+7。多原子イオンなので x + (−2)×4 = −1 → x = +7。
問3H₂O₂ 中の O の酸化数はいくらか。H = +1 とする。
正解はエ：−1。(+1)×2 + x×2 = 0 → x = −1。H₂O₂ の O は例外で −1（通常は −2）。
問4次の反応のうち、酸化還元反応はどれか。
正解はア。Mg：0→+2（酸化）、O：0→−2（還元）と酸化数が変化しているので酸化還元反応。イは酸塩基中和反応、ウは沈殿生成反応、エは化合反応で、いずれも酸化数の変化なし。
問5酸化還元反応において、一方の物質が酸化されると、必ず別の物質が還元される。
○ 正解。電子は「渡す→受け取る」が必ずセット。単独では酸化も還元も起こりません。
問6酸化剤についての記述として正しいものはどれか。
正解はイ。酸化剤は相手から電子を受け取り（自身は還元され）、相手を酸化する物質です。酸化数は減ります。
問7反応 Cl₂ + H₂O₂ → 2HCl + O₂ において、還元剤はどれか。
正解はウ：H₂O₂。H₂O₂ の O：−1 → O₂ の O：0（酸化数が増えた = 酸化された = 還元剤）。Cl₂ の Cl：0 → HCl の Cl：−1（減った = 還元された = 酸化剤）。このときの H₂O₂ は還元剤として機能しています。
問8K₂Cr₂O₇ 中の Cr の酸化数はいくらか。K=+1, O=−2 とする。
正解はイ：+6。x を Cr の酸化数とすると、(+1)×2 + x×2 + (−2)×7 = 0 → 2 + 2x − 14 = 0 → x = +6。
問9KMnO₄ の半反応式（酸性条件）で、Mn の酸化数はいくつ変化するか。
正解はエ：5。KMnO₄ の Mn：+7 → Mn²⁺ の Mn：+2。変化量 = 7 − 2 = 5。よって電子を5個受け取ります（MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O）。
問10全ての物質は酸化剤か還元剤のどちらか一方としてしか機能しない。
× 誤り。H₂O₂ のように二面性を持つ物質があります。KMnO₄ との反応では還元剤、KI との反応では酸化剤として機能します。
