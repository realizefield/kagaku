SO₂・H₂O₂ の「二面性」に注意！

この2つの物質は、反応する相手によって酸化剤にも還元剤にもなります。
入試でも頻出のポイントです。しっかり整理しておきましょう。

❓ なぜ SO₂ は酸化剤にも還元剤にもなるのか

カギは「S の酸化数 +4 が中間値であること」

硫黄（S）の酸化数は −2 〜 +6 の範囲で変化できます。
SO₂ の S は +4 で、ちょうど中間あたりにあります。

物質S の酸化数特徴
H₂S −2（最小） これ以上下がれない → 電子を渡すしかない → 強力な還元剤
SO₂ +4（中間） 上にも下にも動ける → 二面性あり
H₂SO₄（濃） +6（最大） これ以上上がれない → 電子を受け取るしかない → 酸化剤のみ

① SO₂ が還元剤になるとき（強い酸化剤と出会うとき）

KMnO₄・K₂Cr₂O₇・I₂・Cl₂ などの強力な酸化剤と反応するとき、
SO₂ は電子を渡す側（還元剤）に回ります。

SO₂ + 2H₂O → SO₄²⁻ + 4H⁺ + 2e⁻

S の酸化数：+4 → +6（増える → 酸化された → 還元剤）

💡 「強い酸化剤に引っ張られて、S がさらに電子を手放す（+4 → +6）」

② SO₂ が酸化剤になるとき（強い還元剤と出会うとき）

H₂S のような強力な還元剤と反応するとき、
相手（H₂S）の方が電子を渡す力が強いので、
SO₂ は電子を受け取る側（酸化剤）に回ります。

2H₂S + SO₂ → 3S + 2H₂O
物質S の酸化数変化役割
H₂S −2 → 0（増える） 還元剤（酸化された）
SO₂ +4 → 0（減る） 酸化剤（還元された）
💡 H₂S の S（−2）は「これ以上電子を抱えられない」最底辺。
SO₂ の S（+4）より圧倒的に強い還元剤なので、
SO₂ が「やむなく」電子を受け取る側に回る。

まとめ：力関係で決まる

相手力関係SO₂ の役割S の変化
KMnO₄・I₂ など 相手が強い酸化剤 還元剤 +4 → +6
H₂S など 相手が強い還元剤 酸化剤 +4 → 0
一言まとめ：
SO₂ の S は酸化数 +4（中間値） なので上にも下にも動ける。
強い相手に出会ったとき、その相手の逆の役割を引き受ける。

❓ なぜ H₂O₂ は酸化剤にも還元剤にもなるのか

カギは「O の酸化数 −1 が中間値であること」

酸素（O）の酸化数は通常 −2（最小値）ですが、
H₂O₂ の O は −1 という例外的な中間値をとります。

物質O の酸化数特徴
H₂O・OH⁻ −2（最小） これ以上下がれない → 酸化剤にはなれない
H₂O₂ −1（中間） 上（0）にも下（−2）にも動ける → 二面性あり
O₂ 0（最大） これ以上上がれない → 還元剤にはなれない

① H₂O₂ が酸化剤になるとき（ほとんどの場合）

KI・FeSO₄・SO₂ などの一般的な還元剤と反応するとき、
H₂O₂ は電子を受け取る側（酸化剤）になります。

H₂O₂ + 2H⁺ + 2e⁻ → 2H₂O　（酸性）
H₂O₂ + 2e⁻ → 2OH⁻　（中性・塩基性）

O の酸化数：−1 → −2（減る → 還元された → 酸化剤）

💡 O が −1 → −2 に下がる（電子をもらう）のが「普通の H₂O₂」の動き

② H₂O₂ が還元剤になるとき（KMnO₄ との反応のみ）

KMnO₄ は非常に強力な酸化剤（Mn：+7）なので、
H₂O₂ よりも「電子を引き寄せる力」が圧倒的に強く、
H₂O₂ が電子を渡す側（還元剤）に回らざるを得なくなります。

H₂O₂ → O₂ + 2H⁺ + 2e⁻　（還元剤として）

O の酸化数：−1 → 0（増える → 酸化された → 還元剤）

💡 KMnO₄ の「電子を奪う力」が強すぎて、
H₂O₂ の O が −1 → 0 に押し上げられてしまう。

図解：O の酸化数の動き

O の酸化数方向役割生成物
通常 −1 → −2 ⬇ 下がる 酸化剤 H₂O または OH⁻
KMnO₄と反応 −1 → 0 ⬆ 上がる 還元剤 O₂
一言まとめ：
H₂O₂ の O は酸化数 −1（中間値） なので上（0）にも下（−2）にも動ける。
相手が KMnO₄ のような超強力な酸化剤のときだけ、
O が 0 の方向（O₂）に押し上げられて還元剤になる。
それ以外はすべて酸化剤（H₂O または OH⁻ へ）。

❓ なぜ酸性では H₂O、中性・塩基性では OH⁻ になるのか

核心は「H⁺ が使えるかどうか」

H₂O₂ が酸化剤としてはたらくとき、O の酸化数が −1 → −2 に変わります。
この「−2 になった O」が H⁺ を拾えるかどうかで生成物が変わります。

① 酸性条件：H⁺ がたくさんある

H₂O₂ + 2H⁺ + 2e⁻ → 2H₂O

O が e⁻ を受け取って −2 になる
→ 外から H⁺ が 2個 やってくる
→ OH⁻ + H⁺ → H₂O まで一気に進める

💡 酸性 = H⁺ が豊富 → O²⁻ が H⁺ を 2個拾って H₂O になれる

② 中性・塩基性条件：H⁺ がほとんどない

H₂O₂ + 2e⁻ → 2OH⁻

O が e⁻ を受け取って −2 になる
→ 外から H⁺ が来ない
→ H₂O₂ 自身の H をそのまま使う
→ H−O⁻ の形、つまり OH⁻ のまま止まる

💡 中性・塩基性 = H⁺ が不足 → OH⁻ のまま止まる

図で整理

条件H⁺生成物理由
酸性 豊富にある H₂O OH⁻ + H⁺ → H₂O まで進める
中性・塩基性 ほとんどない OH⁻ H⁺ がないので OH⁻ のまま

KMnO₄ も同じ理由

これは H₂O₂ だけでなく KMnO₄ にも当てはまります。

条件KMnO₄ の反応後
酸性Mn²⁺（H⁺ が豊富 → 最後まで還元）
中性・塩基性MnO₂（H⁺ 不足 → 途中で止まる）
一言まとめ：
「OH⁻ に H⁺ を渡せるかどうかの違い」
酸性 → H⁺ がある → H₂O まで行ける
中性・塩基性 → H⁺ がない → OH⁻ のまま止まる
